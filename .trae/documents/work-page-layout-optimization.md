## 具体实施步骤

### 步骤1：调整模板结构顺序
- 轮播图 → 搜索框 → 常用功能 → 系统管理
- 删除"常用功能"和"系统管理"的 `section-title` 标题栏，改为卡片内置标题行（更简约）

### 步骤2：新增搜索框
- 位置：轮播图下方，独立白色卡片内
- 样式：圆角输入框，`#F5F7FA` 浅灰背景，左侧搜索图标（`u-icon name="search"`），`#86909C` 色
- placeholder 文字："搜索功能"
- 右侧清空按钮（输入内容时显示）
- 添加 `searchKeyword` ref 变量
- 添加 `filteredGridList` computed，根据关键词过滤 gridList 的 title 字段

### 步骤3：常用功能区块（上移 + 缩小 + 4个）
- 移到搜索框下方，系统管理上方
- 使用白色卡片包裹，内含标题行"常用功能" + 4个快捷入口
- 标题行样式：左侧文字 `#1D2129` 28rpx font-weight:600，右侧可放"更多"链接
- 4个快捷入口横排等分，参考首页 QuickMenu 风格：
  - 图标背景：`#E8F0FE` 圆形 72rpx
  - 图标颜色：`#3D6DF7`，size="18"
  - 文字：`#1D2129` 22rpx font-weight:500
  - 间距 gap: 12rpx
- 补充第4个功能项："日志查询"，icon 用 `file-text`
- 卡片样式：`border-radius: 20rpx`，`box-shadow: 0 2rpx 12rpx rgba(61, 109, 247, 0.06)`

### 步骤4：系统管理区块样式升级
- 使用白色卡片包裹，内含标题行"系统管理"
- 标题行同上风格
- 网格图标背景色从 `#3c96f3` 改为 `#3D6DF7`（对齐主品牌色）
- 图标尺寸、间距保持不变
- 使用 `filteredGridList` 替代 `gridList` 渲染，实现搜索过滤
- 无搜索结果时显示空状态提示

### 步骤5：全局样式统一
- 页面底色保持 `#F5F7FA`
- 所有卡片统一 `border-radius: 20rpx`
- 所有卡片统一 `box-shadow: 0 2rpx 12rpx rgba(61, 109, 247, 0.06)`
- 交互反馈统一：`transform: scale(0.95)` + `opacity: 0.8` + `transition: 0.2s ease`
- 文字色阶统一使用 `#1D2129` / `#4E5969` / `#86909C`
