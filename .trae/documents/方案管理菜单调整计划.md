### 方案：修改现有菜单指向新页面

### 需要执行的SQL

```sql
-- 修改方案管理菜单的组件路径
UPDATE sys_menu 
SET component = 'business/planList/index',
    update_time = NOW()
WHERE menu_id = 2076;
```
