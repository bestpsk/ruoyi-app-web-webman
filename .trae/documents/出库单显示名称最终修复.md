## 如果重启后仍无效

### 备选方案：在前端添加调试信息

在 `handleView()` 函数中添加 console.log：

```javascript
function handleView(row) { 
  reset(); 
  getStockOut(row.stockOutId).then(response => { 
    console.log('API Response:', response);  // 调试日志
    form.value = response; 
    if (!form.value.items) form.value.items = []; 
    isView.value = true; 
    dialogTitle.value = "查看出库单"; 
    open.value = true 
  }) 
}
```

然后在浏览器控制台（F12）查看返回的数据结构。
