# 企业管理与方案管理合并方案

## 用户需求

在企业管理列表中：
1. **去掉**：年业绩、创建时间
2. **加上**：方案数量、查看方案、开方案操作

## 实现方案

### 修改企业管理页面 (`enterprise/index.vue`)

#### 表格列调整

| 原列 | 操作 | 新列 | 说明 |
|------|------|------|------|
| 企业名称 | 保留 | 企业名称 | - |
| 老板姓名 | 保留 | 老板姓名 | - |
| 联系电话 | 保留 | 联系电话 | - |
| 企业类型 | 保留 | 企业类型 | - |
| 门店数量 | 保留 | 门店数量 | - |
| 年业绩 | **删除** | - | - |
| 企业级别 | 保留 | 企业级别 | - |
| 服务人 | 保留 | 服务人 | - |
| 开始合作 | 保留 | 开始合作 | - |
| 结束合作 | 保留 | 结束合作 | - |
| 状态 | 保留 | 状态 | - |
| 创建时间 | **删除** | - | - |
| - | **新增** | 方案数量 | 显示该企业的方案数量，可点击查看 |
| 操作 | 修改 | 操作 | 查看/开方案 + 修改/删除企业 |

#### 操作列调整

**原操作**：修改、删除

**新操作**：
- 查看方案（弹窗显示方案列表）
- 开方案（弹窗新增方案表单）
- 修改（修改企业信息）
- 删除（删除企业）

### 后端调整

需要在企业列表查询时关联统计方案数量。

修改 `BizEnterpriseService.php` 的 `selectEnterpriseList` 方法，添加方案数量统计。

### 文件修改清单

| 文件路径 | 操作 | 说明 |
|---------|------|------|
| `front/src/views/business/enterprise/index.vue` | 修改 | 调整表格列和操作按钮 |
| `webman/app/service/BizEnterpriseService.php` | 修改 | 添加方案数量统计 |

## 优点

1. **改动最小**：只需修改企业管理页面
2. **信息集中**：企业信息和方案操作在同一页面
3. **操作便捷**：直接在列表中查看方案数量、开方案
4. **保留原有功能**：企业管理的增删改查功能不变
