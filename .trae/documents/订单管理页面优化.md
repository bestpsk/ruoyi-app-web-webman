# 订单管理页面优化方案

## 📋 需求

1. **去掉表格中的"订单状态"列**
2. **搜索条件改为一行显示**

## 📝 具体修改

### 文件：[d:\fuchenpro\front\src\views\business\order\index.vue](file:///d:\fuchenpro\front\src\views\business\order\index.vue)

### 修改1：删除表格中的"订单状态"列（第98-102行）

删除以下代码：
```vue
<el-table-column label="订单状态" align="center" prop="orderStatus" width="90">
  <template #default="scope">
    <el-tag :type="..." size="small">{{ getOrderStatusName(scope.row.orderStatus) }}</el-tag>
  </template>
</el-table-column>
```

### 修改2：删除或保留搜索表单中的"订单状态"

**选项A**：完全删除搜索条件中的"订单状态"（第32-40行）

**选项B**：保留但缩小宽度

建议选择 **选项A**（既然表格都不显示了，搜索也没必要了）

### 修改3：优化搜索条件布局为一行

当前搜索条件：
```
第一行: 订单编号 | 客户姓名 | 企业 | 门店 | 订单状态 | 企业审核 | 财务审核
第二行: 创建时间 | 搜索 | 重置
```

优化后：
```
一行: 订单编号 | 客户姓名 | 企业 | 门店 | 企业审核 | 财务审核 | 创建时间 | 搜索 | 重置
```

**调整策略**：
- 删除"订单状态"搜索项
- 缩小各输入框/选择框宽度
- 创建日期选择器缩小到180px
