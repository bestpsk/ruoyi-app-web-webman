# 查看出库单显示数字修复计划

## 问题根因

**不是后端问题！是前端表单字段绑定错误！**

### 当前代码（stockOut/index.vue）

```html
<!-- 第91行：出库企业 -->
<el-select v-model="form.enterpriseId" ... :disabled="isView">
  <el-option :value="item.enterpriseId" />  <!-- value是ID -->
</el-select>

<!-- 第97行：对接员工 -->  
<el-select v-model="form.contactEmployeeId" ... :disabled="isView">
  <el-option :value="item.userId" />       <!-- value是ID -->
</el-select>
```

**当 isView=true 时：**
- el-select 被禁用
- 显示的值 = `v-model` 的值 = **ID数字**（2, 100）
- 而不是对应的 label（名称）

### 打印预览为什么正确？

第209-211行用的是纯文本绑定 Name 字段：
```html
<span>出库企业：</span>{{ form.enterpriseName || '-' }}  ✅ 正确
```

---

## 解决方案

**查看模式下，将 el-select 替换为纯文本显示**

### 修改1：出库企业/对接员工（outTargetType=1 时）
**文件：** [stockOut/index.vue](file:///d:/fuchenpro/front/src/views/wms/stockOut/index.vue) 第88-101行

```html
<!-- 修改前 -->
<el-col :span="6" v-if="form.outTargetType === '1'">
  <el-form-item label="出库企业" prop="enterpriseId">
    <el-select v-model="form.enterpriseId" ... :disabled="isView">...</el-select>
  </el-form-item>
</el-col>
<el-col :span="6" v-if="form.outTargetType === '1'">
  <el-form-item label="对接员工">
    <el-select v-model="form.contactEmployeeId" ... :disabled="isView">...</el-select>
  </el-form-item>
</el-col>

<!-- 修改后 -->
<el-col :span="6" v-if="form.outTargetType === '1'">
  <el-form-item label="出库企业" prop="enterpriseId">
    <template v-if="!isView">
      <el-select v-model="form.enterpriseId" filterable remote ...>...</el-select>
    </template>
    <span v-else style="line-height: 32px">{{ form.enterpriseName || '-' }}</span>
  </el-form-item>
</el-col>
<el-col :span="6" v-if="form.outTargetType === '1'">
  <el-form-item label="对接员工">
    <template v-if="!isView">
      <el-select v-model="form.contactEmployeeId" filterable remote ...>...</el-select>
    </template>
    <span v-else style="line-height: 32px">{{ form.contactEmployeeName || '-' }}</span>
  </el-form-item>
</el-col>
```

### 修改2：领用员工/目标企业（outTargetType=2 时）
**同上方式处理 responsibleId/responsibleName 和 enterpriseId/enterpriseName**
