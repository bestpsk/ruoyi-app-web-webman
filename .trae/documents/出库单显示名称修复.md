# 查看出库单企业和员工显示名称修复计划

## 问题描述
查看出库单时，出库企业显示 "2"（ID），对接员工显示 "100"（ID），而不是实际名称。

## 问题分析

### 当前代码逻辑（已修改但未生效）
```php
$enterpriseName = $stockOutArray['enterprise_name'];
if (empty($enterpriseName) && !empty($stockOutArray['enterprise_id'])) {
    // 查询企业名称...
}
```

### 为什么没有生效？

**根本原因**：数据库中 `enterprise_name`、`contact_employee_name` 等字段存储的**就是 ID 值**（如 "2"、"100"），而不是空字符串。

- `$stockOutArray['enterprise_name'] = "2"`  （存储的是ID）
- `empty("2")` → **false**（非空字符串）
- 条件不满足，不会执行查询名称的逻辑

### 需要的修复方案

**判断条件改为**：检查字段值是否为**纯数字**（ID），如果是则查询实际名称。

**修改后**:
```php
// 处理企业名称：如果值为纯数字或空，则查询真实名称
$enterpriseName = $stockOutArray['enterprise_name'] ?? '';
if ((is_numeric($enterpriseName) || empty($enterpriseName)) && !empty($stockOutArray['enterprise_id'])) {
    $enterprise = BizEnterprise::find($stockOutArray['enterprise_id']);
    if ($enterprise) {
        $enterpriseName = $enterprise->enterprise_name;
    } else {
        $enterpriseName = $stockOutArray['enterprise_name'] ?? '-';
    }
}

// 处理对接员工姓名：如果值为纯数字或空，则查询真实姓名
$contactEmployeeName = $stockOutArray['contact_employee_name'] ?? '';
if ((is_numeric($contactEmployeeName) || empty($contactEmployeeName)) && !empty($stockOutArray['contact_employee_id'])) {
    $user = SysUser::find($stockOutArray['contact_employee_id']);
    if ($user) {
        $contactEmployeeName = $user->real_name ?: $user->user_name;
    } else {
        $contactEmployeeName = $stockOutArray['contact_employee_name'] ?? '-';
    }
}

// 处理领用员工姓名：如果值为纯数字或空，则查询真实姓名
$responsibleName = $stockOutArray['responsible_name'] ?? '';
if ((is_numeric($responsibleName) || empty($responsibleName)) && !empty($stockOutArray['responsible_id'])) {
    $user = SysUser::find($stockOutArray['responsible_id']);
    if ($user) {
        $responsibleName = $user->real_name ?: $user->user_name;
    } else {
        $responsibleName = $stockOutArray['responsible_name'] ?? '-';
    }
}
```

## 关键改动说明

| 判断条件 | 修改前 | 修改后 |
|---------|--------|--------|
| 企业名称 | `empty($name)` | `is_numeric($name) \|\| empty($name)` |
| 对接员工 | `empty($name)` | `is_numeric($name) \|\| empty($name)` |
| 领用员工 | `empty($name)` | `is_numeric($name) \|\| empty($name)` |

**原因**：
- `"2"` 是非空字符串 → `empty("2")` = **false**
- `is_numeric("2")` = **true** ✅ 能正确识别为 ID
