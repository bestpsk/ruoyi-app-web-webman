# 入库单打印预览和查看详情优化计划

## 问题1：打印预览太窄，内容显示不完全

### 当前状态
- 打印弹窗宽度：`width="900px"`（第181行）
- 打印容器宽度：`190mm`（约720px）（第606行）
- 表格列使用**固定像素宽度**：
  - 序号：40px
  - 供货商：80px
  - 规格：50px
  - 换算：80px
  - **数量：90px** — 显示 `10盒（100支）` 被挤成两行 ❌
  - **单价：70px** — 太窄 ❌
  - 金额：80px

### 截图问题
- 数量列 `10盒（100支）` 换行显示为两行
- 合计行的 `10盒（100支）` 也被挤成两行
- 整体表格内容拥挤

### 修复方案
1. 打印弹窗宽度加大到 `1100px`
2. 表格列改用 **百分比或更大最小宽度**
3. 数量列和单价列加宽

## 问题2：查看入库单 — 数量和单价缺少单位

### 当前状态（第134-144行）

```vue
<!-- 数量列 -->
<span v-else>{{ scope.row.quantity }}</span>    <!-- 只显示 100 -->

<!-- 进货单价列 -->
<span v-else>{{ scope.row.purchasePrice }}</span>  <!-- 只显示 258 -->
```

### 期望效果
- 数量列：`100支` 或 `10盒（100支）`
- 进货单价列：`258/支` 或 `2580/盒`

### 修复方案
- 数量列：使用 `formatItemQuantity(item)` 方法（已有）
- 进货单价列：添加 `formatItemPrice(item)` 方法，格式为 `价格/单位`

## 实施步骤

### 步骤1：修改打印预览弹窗和表格

**打印弹窗宽度**：`900px` → `1100px`

**表格列宽调整**：

| 列名 | 旧宽度 | 新宽度 |
|------|--------|--------|
| 序号 | 40 | 45 |
| 货品名称 | auto | min-width: 120 |
| 供货商 | 80 | min-width: 90 |
| 规格 | 50 | min-width: 55 |
| 换算 | 80 | min-width: 90 |
| 数量 | 90 | **min-width: 130** |
| 单价 | 70 | **min-width: 90** |
| 金额 | 80 | min-width: 95 |

### 步骤2：修改查看入库单明细表格

**数量列**（第137行）：
```vue
<!-- 旧 -->
<span v-else>{{ scope.row.quantity }}</span>
<!-- 新 -->
<span v-else>{{ formatItemQuantity(scope.row) }}</span>
```

**进货单价列**（第143行）：
```vue
<!-- 旧 -->
<span v-else>{{ scope.row.purchasePrice }}</span>
<!-- 新 -->
<span v-else>{{ formatItemPrice(scope.row) }}</span>
```

### 步骤3：添加 formatItemPrice 方法

```javascript
function formatItemPrice(item) {
  const unitLabel = item.unitType === '1' ? getUnitLabel(item.unit) : getSpecLabel(item.spec)
  return item.purchasePrice + '/' + (unitLabel || '')
}
```
