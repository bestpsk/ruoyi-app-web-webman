# 行程安排用户姓名和职位显示优化计划

## 一、需求分析

### 1.1 用户需求
1. **用户管理**：将"用户昵称"改为"用户姓名"
2. **行程安排**：员工姓名使用用户管理的"用户姓名"（nick_name）而非登录账号（user_name）
3. **日历视图**：职位信息以tag形式显示在员工姓名下方
4. **排班列表**：增加一列职位

### 1.2 当前问题分析

| 问题 | 当前状态 | 期望状态 |
|------|---------|---------|
| 用户昵称标签 | 显示"用户昵称" | 显示"用户姓名" |
| 员工姓名来源 | 使用`user_name`(登录账号如"admin") | 使用`nick_name`(用户姓名如"若依") |
| 日历视图职位 | 已有代码但数据未正确获取 | 正确显示职位标签 |
| 排班列表职位 | 显示在员工列下方 | 新增独立职位列 |

### 1.3 数据库字段说明
- `sys_user.user_name` - 登录账号（如 admin, ry）
- `sys_user.nick_name` - 用户姓名/昵称（如 若依, 测试）
- `sys_post.post_name` - 岗位名称（如 董事长, 测试员）
- `sys_user_post` - 用户与岗位关联表

### 修复2：后端 - 员工行程使用nick_name并关联职位 ⏱️ 10分钟

**文件**：`d:\fuchenpro\webman\app\service\BizScheduleService.php`

**修改`selectEmployeeSchedule`方法**：
```php
public function selectEmployeeSchedule($params)
{
    // ... 现有查询逻辑 ...
    
    $result = [];
    foreach ($users as $user) {
        $userSchedules = $schedules->where('user_id', $user->user_id);
        $scheduleMap = [];
        
        foreach ($userSchedules as $schedule) {
            $day = date('j', strtotime($schedule->schedule_date));
            $scheduleMap[$day] = $schedule;
        }
        
        // 获取用户职位
        $postInfo = Db::table('sys_user_post up')
            ->join('sys_post p', 'up.post_id', '=', 'p.post_id')
            ->where('up.user_id', $user->user_id)
            ->first();
        
        $result[] = [
            'user_id' => $user->user_id,
            'user_name' => $user->nick_name ?? $user->user_name,  // 使用用户姓名
            'post_name' => $postInfo->post_name ?? '',
            'schedules' => $scheduleMap
        ];
    }
    
    return $result;
}
```

**同样修改`selectEnterpriseSchedule`方法**，确保企业排班视图也使用正确的姓名。

### 修复3：后端 - 排班列表数据增加职位信息 ⏱️ 5分钟

**文件**：`d:\fuchenpro\webman\app\service\BizScheduleService.php`

**修改`selectScheduleList`方法**，关联查询职位：
```php
public function selectScheduleList($params = [])
{
    $query = BizSchedule::query();
    // ... 现有筛选条件 ...
    
    $result = $query->orderBy('schedule_date', 'desc')
        ->orderBy('schedule_id', 'desc')
        ->paginate($pageSize, ['*'], 'page', $pageNum);
    
    // 补充用户姓名和职位信息
    foreach ($result->items() as $item) {
        $user = SysUser::find($item->user_id);
        if ($user) {
            $item->user_name = $user->nick_name ?? $user->user_name;
            
            // 获取职位
            $postInfo = Db::table('sys_user_post up')
                ->join('sys_post p', 'up.post_id', '=', 'p.post_id')
                ->where('up.user_id', $user->user_id)
                ->first();
            $item->post_name = $postInfo->post_name ?? '';
        }
    }
    
    return $result;
}
```
