# 多货品入库单 — 总数量显示优化方案

## 问题分析

当入库单包含多个**不同规格**的货品时：
- 货品A：1盒=10支
- 货品B：1套=5个
- 无法用统一单位表示总数量

## 推荐方案：按货品种类数区分

### 显示逻辑

| 场景 | 显示格式 | 示例 |
|------|----------|------|
| **单个货品** | 大单位（小单位） | `10盒（100支）` |
| **多个同规格货品** | 大单位（小单位）汇总 | `20盒（200支）` |
| **多个不同规格货品** | 种类数 + 纯数字 | `2种 / 120` |

### 效果预览

```
入库单号    总数量              总金额     操作
RK001      10盒（100支）        25800.00   查看    ← 单个货品
RK002      2种 / 120           51800.00   查看    ← 多个不同规格
RK003      3盒（30支）          7740.00    查看    ← 多个同规格
```

### 判断"是否同规格"的逻辑

比较所有 items 的 `unit` 和 `spec` 字段：
- 如果所有 item 的 unit 和 spec 都相同 → 同规格 → 可以汇总显示
- 如果有不同的 unit 或 spec → 不同规格 → 只显示种类数+纯数字

## 实施步骤

修改 `formatQuantity` 方法：

```javascript
function formatQuantity(row) {
  const items = row.items || []
  
  if (items.length === 0) {
    return row.totalQuantity || 0
  }
  
  if (items.length === 1) {
    // 单个货品：原有格式
    const item = items[0]
    const packQty = row.display_pack_qty || item.packQty || 1
    const unitLabel = getUnitLabel(row.display_unit || item.unit)
    const specLabel = getSpecLabel(row.display_spec || item.spec)
    const totalQty = row.totalQuantity || 0
    if (packQty > 1 && specLabel) {
      const mainQty = totalQty / packQty
      const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
      return mainQtyStr + unitLabel + '（' + totalQty + specLabel + '）'
    } else if (unitLabel) {
      return totalQty + unitLabel
    } else {
      return totalQty
    }
  }
  
  // 多个货品：检查是否同规格
  const firstUnit = items[0].unit
  const firstSpec = items[0].spec
  const isSameSpec = items.every(item => item.unit === firstUnit && item.spec === firstSpec)
  
  if (isSameSpec) {
    // 同规格：可以汇总显示
    const packQty = items[0].packQty || 1
    const unitLabel = getUnitLabel(firstUnit)
    const specLabel = getSpecLabel(firstSpec)
    const totalQty = row.totalQuantity || 0
    if (packQty > 1 && specLabel) {
      const mainQty = totalQty / packQty
      const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
      return mainQtyStr + unitLabel + '（' + totalQty + specLabel + '）'
    } else {
      return totalQty + (specLabel || unitLabel || '')
    }
  } else {
    // 不同规格：显示种类数 + 纯数字
    return items.length + '种 / ' + (row.totalQuantity || 0)
  }
}
```
