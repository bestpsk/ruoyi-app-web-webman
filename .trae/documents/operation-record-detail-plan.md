# 操作记录详情展示方案

## 问题
操作记录表格中缺少顾客反馈、操作前后对比照、备注的展示，这些信息在调阅记录时很重要。

## 方案：表格行展开（expand row）

在操作记录表格中增加**可展开行**，点击行左侧的展开箭头，展开显示详情：

```
┌──┬──────┬──────┬────┬─────┬────┬──────┬──────┬──────┐
│▶ │ 类型  │ 编号  │ 品项 │ 次数 │ 金额 │ 操作人 │ 满意度 │ 日期  │
├──┼──────┼──────┼────┼─────┼────┼──────┼──────┼──────┤
│▼ │ 持卡  │ PK.. │ 美白 │  1  │ 200 │ 张三  │ ★★★★ │ 05-03│
├──┴──────┴──────┴────┴─────┴────┴──────┴──────┴──────┤
│  顾客反馈：皮肤比上次好很多                            │
│  操作前：[照片1] [照片2]    操作后：[照片1] [照片2]      │
│  备注：下次加强补水                                    │
├──┬──────┬──────┬────┬─────┬────┬──────┬──────┬──────┤
│▶ │ 体验  │  -   │ 祛斑 │  1  │ 100 │ 李四  │ ★★★  │ 05-02│
└──┴──────┴──────┴────┴─────┴────┴──────┴──────┴──────┘
```

## 修改文件

### `d:\fuchenpro\front\src\views\business\sales\index.vue`

1. 操作记录表格增加 `type="expand"` 列
2. 展开行中显示：
   - 顾客反馈（文本）
   - 操作前对比照（image-upload 只读模式 / el-image 预览）
   - 操作后对比照（同上）
   - 备注（文本）
3. 使用 `el-image` 组件支持图片预览（点击放大）
