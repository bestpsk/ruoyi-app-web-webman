# 出库单逻辑检查与修复计划

## 发现的问题

### 🔴 严重问题：confirmStockOut 存在双重换算 Bug

**问题描述**：
- `insertStockOut` 已将 quantity 换算为最小单位存储（如：1盒=10支 → 存储quantity=10）
- 但 `confirmStockOut` 在确认出库时 **又做了一次换算**

```php
// confirmStockOut 第176-189行
foreach ($items as $item) {
    $product = BizProduct::find($item->product_id);
    $packQty = $product ? ($product->pack_qty ?? 1) : 1;
    $unitType = $item->unit_type ?? '1';
    $actualQty = $item->quantity;           // ← 此时已是换算后的值(如10)
    if ($unitType === '1') {
        $actualQty = $item->quantity * $packQty;  // ← 又乘以10！变成100！
    }
}
```

**影响**：
| 场景 | 用户输入 | 数据库存 | 确认时计算 | 实际扣减 |
|------|---------|----------|-----------|---------|
| 出库1盒 | 1盒 | 10支 | 10×10=**100支** | ❌ 错误！ |

### 🟡 中等问题：selectStockOutById 缺少字段转换

**问题描述**：
- 入库单的 `selectStockInById` 返回了 `originalQuantity` 字段
- 出库单的 `selectStockOutById` 直接返回原始数据，没有返回 `originalQuantity`

**影响**：查看/修改出库单详情时，可能无法正确显示原始输入的数量

## 修复方案

### 步骤1：修复 confirmStockOut 方法（关键）

由于保存时已经是最小单位数量，确认出库时 **不需要再做换算**：

```php
public function confirmStockOut($stockOutId)
{
    // ...验证代码保持不变...

    foreach ($items as $item) {
        // 直接使用 item->quantity（已经是最小单位），不再换算
        $actualQty = $item->quantity;
        
        $inventory = BizInventory::where('product_id', $item->product_id)->first();
        if (!$inventory || $inventory->quantity < $actualQty) {
            $productName = $product ? $product->product_name : $item->product_name;
            $currentQty = $inventory ? $inventory->quantity : 0;
            return ['success' => false, "msg" => "货品【{$productName}】库存不足"];
        }
    }

    foreach ($items as $item) {
        $actualQty = $item->quantity;  // 直接使用，不做换算
        
        $inventory = BizInventory::where('product_id', $item->product_id)->first();
        $inventory->quantity -= $actualQty;
        $inventory->save();
    }
}
```

### 步骤2：修复 selectStockOutById 方法

添加字段转换，返回 `originalQuantity`：

```php
public function selectStockOutById($stockOutId)
{
    $stockOut = BizStockOut::find($stockOutId);
    if ($stockOut) {
        $items = BizStockOutItem::where('stock_out_id', $stockOutId)->get()->toArray();
        $stockOut->items = array_map(function ($item) {
            return [
                'itemId' => $item['id'] ?? null,
                'productId' => $item['product_id'],
                'productName' => $item['product_name'],
                'spec' => $item['spec'],
                'unit' => $item['unit'],
                'unitType' => $item['unit_type'] ?? '1',
                'packQty' => $item['pack_qty'] ?? 1,
                'originalQuantity' => $item['original_quantity'] ?? $item['quantity'],
                'quantity' => $item['quantity'],
                'salePrice' => floatval($item['sale_price']),
                'amount' => $item['amount'],
                'remark' => $item['remark'],
            ];
        }, $items);
        
        // 制单人姓名处理...
    }
    return $stockOut;
}
```
