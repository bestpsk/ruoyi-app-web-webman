# 重新设计按钮权限管理方案

## 问题分析

之前的方案存在根本问题：
1. 通过 `setChecked` 操作 el-tree 会影响父节点的选中状态（父子联动）
2. watch 双向同步导致循环触发
3. 按钮权限和菜单权限混在一起管理

## 新方案：完全独立管理按钮权限

### 核心思路

1. **菜单权限**：由 el-tree 管理（勾选菜单节点）
2. **按钮权限**：由 `buttonCheckState` 独立管理（不通过 el-tree）
3. **提交时合并**：将两者合并后提交

### 实施步骤

**文件**: `front/src/views/system/role/index.vue`

#### 步骤1：删除两个 watch

删除现有的两个 watch，不再同步。

#### 步骤2：修改提交函数

```javascript
function submitForm() {
  proxy.$refs["roleRef"].validate(valid => {
    if (valid) {
      // 获取菜单选中项
      const menuKeys = menuRef.value.getCheckedKeys()
      const halfKeys = menuRef.value.getHalfCheckedKeys()
      
      // 获取按钮权限选中项
      const buttonKeys = Object.keys(buttonCheckState.value)
        .filter(id => buttonCheckState.value[id])
        .map(id => Number(id))
      
      // 合并
      form.value.menuIds = [...menuKeys, ...halfKeys, ...buttonKeys]
      
      if (form.value.roleId != undefined) {
        updateRole(form.value).then(() => {
          proxy.$modal.msgSuccess("修改成功")
          open.value = false
          getList()
        })
      } else {
        addRole(form.value).then(() => {
          proxy.$modal.msgSuccess("新增成功")
          open.value = false
          getList()
        })
      }
    }
  })
}
```

#### 步骤3：修改加载函数

在 `handleUpdate` 中初始化 `buttonCheckState`：

```javascript
function handleUpdate(row) {
  reset()
  const roleId = row.roleId || ids.value
  const roleMenu = getRoleMenuTreeselect(roleId)
  getRole(roleId).then(response => {
    form.value = response.data
    form.value.roleSort = Number(form.value.roleSort)
    open.value = true
    nextTick(() => {
      roleMenu.then((res) => {
        // 分离菜单ID和按钮ID
        const menuIds = []
        const buttonIds = []
        res.checkedKeys.forEach(id => {
          const node = findNodeById(menuOptions.value, id)
          if (node) {
            if (!node.children || node.children.length === 0) {
              buttonIds.push(id)
            } else {
              menuIds.push(id)
            }
          }
        })
        
        // 设置菜单选中
        menuIds.forEach(v => {
          nextTick(() => {
            menuRef.value.setChecked(v, true, false)
          })
        })
        
        // 设置按钮选中状态
        const btnState = {}
        buttonIds.forEach(id => {
          btnState[id] = true
        })
        buttonCheckState.value = btnState
      })
    })
  })
  title.value = "修改角色"
}
```

```javascript
function reset() {
  if (menuRef.value != undefined) {
    menuRef.value.setCheckedKeys([])
  }
  menuExpand.value = false
  menuNodeAll.value = false
  deptExpand.value = true
  deptNodeAll.value = false
  buttonCheckState.value = {}  // 重置按钮状态
  form.value = {
    roleId: undefined,
    roleName: undefined,
    roleKey: undefined,
    roleSort: 0,
    status: "0",
    menuIds: [],
    deptIds: [],
    menuCheckStrictly: true,
    deptCheckStrictly: true,
    remark: undefined
  }
  proxy.resetForm("roleRef")
}
```
