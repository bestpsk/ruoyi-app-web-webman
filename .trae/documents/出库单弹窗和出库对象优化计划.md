# 出库单弹窗调整和出库对象优化计划

## 需求分析

1. **弹窗宽度调宽** - 当前宽度 `1050px`，需要适当调宽
2. **出库对象优化** - 出库可能出给企业，也可能出给个人（操作的员工），员工带到企业的门店使用

## 当前结构

### 前端表单字段（stockOut/index.vue）

* 出库类型

* 出库企业（enterpriseId，下拉选择）

* 负责员工（responsibleName，文本输入）

* 出库日期

* 备注

### 后端模型字段（BizStockOut.php）

* enterprise\_id, enterprise\_name - 出库企业

* responsible\_id, responsible\_name - 负责员工

## 解决方案

### 方案设计：添加"出库对象类型"字段

**出库对象类型**：

* **企业出库** - 直接出库给企业

* **员工领用** - 员工领用后带到企业门店使用

**表单逻辑**：

1. 选择"企业出库"：显示企业选择框
2. 选择"员工领用"：显示员工选择框 + 目标企业选择框（可选）

### 字段调整

| 字段                            | 说明                          |
| ----------------------------- | --------------------------- |
| outTargetType                 | 出库对象类型：1-企业出库，2-员工领用        |
| enterpriseId/enterpriseName   | 出库企业（企业出库时必填，员工领用时可选填写目标企业） |
| responsibleId/responsibleName | 领用员工（员工领用时必填）               |
