# 入库管理列表 — 显示入库货品信息优化方案

## 问题

入库管理列表看不到具体入库了什么货品，只能点击"查看"按钮才能看到详情。

## 解决方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **方案1：直接显示列** | 简单直观 | 多货品时显示不全 | ⭐⭐⭐ |
| **方案2：展开行** | 可查看完整明细 | 需要点击展开 | ⭐⭐⭐⭐⭐ |
| **方案3：Popover悬停** | 不占空间 | 移动端不友好 | ⭐⭐⭐ |
| **方案4：标签展示** | 一目了然 | 多货品时标签太多 | ⭐⭐⭐⭐ |

## 推荐方案：方案2 + 方案4 结合

**使用 Element Plus 的展开行功能 + 货品名称列显示标签**

### 效果预览

```
▼ 入库单号   货品名称           供货商      总数量      总金额   ...
▶ RK001     [GCS-p7][测试货品]  南京伊美荟  10盒(100支) 25800   ...

点击展开后：
▼ 入库单号   货品名称           供货商      总数量      总金额   ...
▼ RK001     [GCS-p7][测试货品]  南京伊美荟  10盒(100支) 25800   ...
  ┌─────────────────────────────────────────────────────────────┐
  │ 货品名称   规格   换算      数量       单价    金额         │
  │ GCS-p7    支    1盒=10支  10盒(100支) 258/支  25800       │
  │ 测试货品   个    -        5个        100/个  500         │
  └─────────────────────────────────────────────────────────────┘
```

## 实施步骤

### 步骤1：添加展开行功能

使用 `el-table` 的 `type="expand"` 列：

```vue
<el-table v-loading="loading" :data="stockInList" style="width: 100%">
  <el-table-column type="expand">
    <template #default="scope">
      <el-table :data="scope.row.items" border style="margin: 10px 20px">
        <el-table-column label="货品名称" prop="productName" min-width="120" />
        <el-table-column label="规格" min-width="60" align="center">
          <template #default="itemScope">
            {{ itemScope.row.unitType === '1' ? getUnitLabel(itemScope.row.unit) : getSpecLabel(itemScope.row.spec) }}
          </template>
        </el-table-column>
        <el-table-column label="换算" min-width="80" align="center">...</el-table-column>
        <el-table-column label="数量" min-width="100" align="center">...</el-table-column>
        <el-table-column label="单价" min-width="80" align="right">...</el-table-column>
        <el-table-column label="金额" min-width="80" align="right">...</el-table-column>
      </el-table>
    </template>
  </el-table-column>
  ...其他列
</el-table>
```

### 步骤2：添加货品名称列（标签形式）

在入库单号后面添加货品名称列，使用 `el-tag` 显示：

```vue
<el-table-column label="货品名称" min-width="140" align="center">
  <template #default="scope">
    <template v-if="scope.row.items && scope.row.items.length > 0">
      <el-tag v-for="(item, idx) in scope.row.items.slice(0, 2)" :key="idx" size="small" style="margin: 2px">
        {{ item.productName }}
      </el-tag>
      <el-tag v-if="scope.row.items.length > 2" size="small" type="info" style="margin: 2px">
        +{{ scope.row.items.length - 2 }}
      </el-tag>
    </template>
  </template>
</el-table-column>
```

### 步骤3：添加供货商列

```vue
<el-table-column label="供货商" min-width="100" align="center">
  <template #default="scope">
    <span>{{ scope.row.items?.[0]?.supplierName || '-' }}</span>
  </template>
</el-table-column>
```

### 步骤4：确保 getList 已加载 items 数据

当前 `getList` 已通过 `getStockIn(row.stockInId)` 获取详情并补全 display 字段，同时也有 items 数据。
