# 入库单状态按钮修复计划

## 问题描述
1. **状态按钮不能被点击** - `confirmStockInById` 和 `cancelConfirmStockIn` 方法未在 import 语句中导入
2. **取值反了** - 已确认变成灰色，未确认显示主色（蓝色）

## 问题分析

### 问题1：按钮无法点击
- **文件**: [index.vue:253](front/src/views/wms/stockIn/index.vue#L253)
- **原因**: import 语句只导入了 `confirmStockIn`，但没有导入 `confirmStockInById` 和 `cancelConfirmStockIn`
- **影响**: 调用 `handleStatusChange` 时会报错 `confirmStockInById is not defined`

### 问题2：取值反了
- **文件**: [index.vue:54](front/src/views/wms/stockIn/index.vue#L54)
- **当前代码**: `active-value="1" inactive-value="0"`
- **后端逻辑**:
  - 新建入库单时 status='0'（未确认）
  - 确认入库单后 status='1'（已确认）
- **预期行为**: 
  - status='1'（已确认）→ 显示蓝色（激活状态）
  - status='0'（未确认）→ 显示灰色（未激活状态）
- **实际现象**: 用户反馈相反，需要验证或调整

### 问题3：缺少取消确认的后端接口
- **文件**: [BizStockInController.php](webman/app/controller/wms/BizStockInController.php)
- **现状**: 只有 `confirm` 方法，没有 `cancelConfirm` 方法
- **影响**: 前端调用 `cancelConfirmStockIn` 会返回 404

### 步骤2：添加后端取消确认接口

```php
public function cancelConfirm(Request $request)
{
    $parts = explode('/', $request->path());
    $id = intval(end($parts));
    $service = new BizStockInService();
    $result = $service->cancelConfirmStockIn($id);
    if (!$result['success']) return AjaxResult::error($result['msg']);
    return AjaxResult::success($result['msg']);
}
```

### 步骤3：添加后端取消确认服务方法
**文件**: `d:\fuchenpro\webman\app\service\BizStockInService.php`
**位置**: 在 `confirmStockIn` 方法后添加

**新增方法**:
```php
public function cancelConfirmStockIn($stockInId)
{
    $stockIn = BizStockIn::find($stockInId);
    if (!$stockIn) {
        return ['success' => false, 'msg' => '入库单不存在'];
    }
    if ($stockIn->status === '0') {
        return ['success' => false, 'msg' => '入库单未确认，无需取消'];
    }
    
    // 扣减库存
    $items = BizStockInItem::where('stock_in_id', $stockInId)->get();
    foreach ($items as $item) {
        $inventory = BizInventory::where('product_id', $item->product_id)->first();
        if ($inventory) {
            $actualQty = intval($item->quantity);
            $inventory->quantity = max(0, $inventory->quantity - $actualQty);
            $inventory->update_time = date('Y-m-d H:i:s');
            $inventory->save();
        }
    }
    
    // 更新状态为未确认
    BizStockIn::where('stock_in_id', $stockInId)->update([
        'status' => '0',
        'update_time' => date('Y-m-d H:i:s'),
    ]);
    
    return ['success' => true, 'msg' => '已取消确认'];
}
```
