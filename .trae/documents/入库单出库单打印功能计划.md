# 入库单和出库单打印功能计划

## 需求分析

为入库单和出库单添加打印功能，需要确定：
1. 打印按钮放置位置
2. 打印单据样式设计
3. 技术实现方案

## 一、打印按钮位置

### 方案：在查看弹窗底部添加打印按钮

**位置：查看弹窗底部，关闭按钮左边**

```
┌─────────────────────────────────────────┐
│                                         │
│           [查看详情内容]                 │
│                                         │
├─────────────────────────────────────────┤
│         [打印]  [关闭]                   │
└─────────────────────────────────────────┘
```

**说明**：
- 只在查看模式（isView=true）下显示打印按钮
- 新增和修改模式不显示打印按钮
- 打印按钮在关闭按钮左边，符合操作习惯

## 三、技术实现方案

### 方案：使用 Vue3 + CSS 打印样式

**优点**：
- 无需额外依赖
- 实现简单
- 样式可控

**实现步骤**：

### 步骤1：创建打印组件
创建两个打印组件：
- `src/views/wms/stockIn/print.vue` - 入库单打印模板
- `src/views/wms/stockOut/print.vue` - 出库单打印模板

### 步骤2：添加打印按钮
在列表页和弹窗中添加打印按钮

### 步骤3：添加打印样式
使用 `@media print` 定义打印专用样式

### 步骤4：实现打印函数
```javascript
function handlePrint(row) {
  // 方案1：新窗口打印
  const printWindow = window.open('', '_blank')
  printWindow.document.write(printTemplate)
  printWindow.print()
  
  // 方案2：当前页面打印（需要隐藏其他元素）
}
```

## 五、打印样式要点

1. **纸张大小**：A4纸 (210mm × 297mm)
2. **页边距**：上下左右各 10mm
3. **字体大小**：正文 12pt，标题 16pt 加粗
4. **表格边框**：1px 实线
5. **签字区域**：留空白供手写签字
