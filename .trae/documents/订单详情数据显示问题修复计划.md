# 订单详情页面数据显示问题修复计划

## 问题描述
订单详情页面中以下信息无法显示：
1. 套餐名称（packageName/package_name）
2. 项目信息（品项、次数、价格、成交金额、单次价、实付、欠款）

## 问题根因分析

### 1. 数据流分析

**前端提交流程** (AppV3/src/pages/business/sales/order.vue:344):
```javascript
await addSalesOrder({
  customerId: customerId.value,
  storeId: storeId.value,
  items: orderItems,  // 格式: [{packageItemId, itemName, count, price}]
  remark: orderRemark.value
})
```

**后端期望格式** (BizSalesOrderService.php:46-93):
```php
// 主表数据应包含:
package_name => '套餐名称'

// 明细数组应包含:
[
  'product_name' => '品项名称',
  'quantity' => 次数,
  'plan_price' => 方案价格,
  'is_deal' => 是否成交,
  'deal_amount' => 成交金额,
  'paid_amount' => 实付金额
]
```

### 2. 核心问题

#### 问题1：字段名不匹配
- 前端发送：`itemName`, `count`, `price`
- 后端期望：`product_name`, `quantity`, `plan_price`

#### 问题2：缺失关键字段
- ❌ 未传递 `package_name` → 导致套餐名称为空
- ❌ 未传递 `is_deal` → 无法判断是否成交
- ❌ 未传递 `deal_amount` → 成交金额为0
- ❌ 未传递 `paid_amount` → 实付金额为0

#### 问题3：数据库字段缺失
- `biz_order_item` 表缺少 `paid_amount` 字段（模型中定义了但表中不存在）

## 修复方案

### 步骤1：数据库结构修复
**文件**: d:\fuchenpro\webman\sql\fix_order_items.sql（新建）

为 `biz_order_item` 表添加缺失的 `paid_amount` 字段：
```sql
ALTER TABLE `biz_order_item`
ADD COLUMN `paid_amount` decimal(12,2) DEFAULT 0.00 COMMENT '实付金额' AFTER `deal_amount`;
```

### 步骤2：后端数据适配层
**文件**: d:\fuchenpro\webman\app\service\BizSalesOrderService.php

修改 `insertOrder()` 方法（第45-93行），添加数据转换逻辑：

```php
public function insertOrder($data, $items = [])
{
    // 生成订单号等原有逻辑...
    
    // 转换items数据格式
    $convertedItems = [];
    foreach ($items as $item) {
        $convertedItem = [
            'order_id' => 0, // 稍后填充
            'product_name' => $item['item_name'] ?? $item['productName'] ?? '',
            'quantity' => intval($item['count'] ?? $item['quantity'] ?? 1),
            'plan_price' => floatval($item['price'] ?? $item['plan_price'] ?? 0),
            'is_deal' => isset($item['is_deal']) ? $item['is_deal'] : '1', // 默认成交
            'deal_amount' => floatval($item['price'] ?? $item['deal_amount'] ?? 0),
            'paid_amount' => floatval($item['paid_amount'] ?? 0), // 新增字段
            'remark' => $item['remark'] ?? null,
            'create_time' => date('Y-m-d H:i:s')
        ];
        $convertedItems[] = $convertedItem;
    }
    
    // 使用转换后的$convertedItems替代原$items
    // ...后续逻辑保持不变
}
```

### 步骤3：前端提交数据完善

async function submitOrder() {
  const orderItems = []
  packageList.value.forEach(pkg => {
    (pkg.items || []).forEach(item => {
      if (item.orderCount > 0) {
        orderItems.push({
          packageItemId: item.packageItemId,
          productName: item.productName,        // 改为productName
          quantity: item.orderCount,             // 改为quantity
          planPrice: item.unitPrice || 0,        // 改为planPrice
          dealPrice: item.unitPrice || 0,        // 新增：成交价
          isDeal: '1',                           // 新增：默认成交
          paidAmount: item.unitPrice || 0,       // 新增：实付金额
          price: item.unitPrice || 0             // 保留兼容
        })
      }
    })
  })
  
  if (orderItems.length === 0) {
    uni.showToast({ title: '请选择开单项目', icon: 'none' })
    return
  }
  
  submitting.value = true
  try {
    // 获取当前选中的套餐名称
    const currentPackageName = packageList.value.length > 0 
      ? packageList.value[0].packageName 
      : ''
    
    await addSalesOrder({
      customerId: customerId.value,
      customerName: customerInfo.value?.customerName || '',
      storeId: storeId.value,
      packageName: currentPackageName,           // 新增：套餐名称
      items: orderItems,
      remark: orderRemark.value
    })
    
    uni.showToast({ title: '开单成功', icon: 'success' })
    orderRemark.value = ''
    loadPackages()
  } catch (e) {
    console.error('开单失败:', e)
  } finally {
    submitting.value = false
  }
}
```

### 步骤4：订单详情页显示优化
**文件**: d:\fuchenpro\AppV3\src\pages\business\order\detail.vue

<view class="items-section" v-if="orderItems.length > 0">

    <view class="item-detail">
      <text>方案价: ¥{{ item.planPrice || item.plan_price || item.price || '0.00' }}</text>
      <text>单次价: ¥{{ getUnitPrice(item) }}</text>
    </view>

      <text>成交: ¥{{ item.dealAmount || item.deal_amount || '0.00' }}</text>

    </view>
    
    <view class="item-detail" v-if="(item.dealAmount || item.deal_amount || 0) - (item.paidAmount || item.paid_amount || 0) > 0">
      <text class="owed-amount">欠款: ¥{{ getOwedAmount(item) }}</text>
    </view>
  </view>
</view>
```

添加辅助函数：

```javascript
function getOwedAmount(item) {
  const deal = Number(item.dealAmount || item.deal_amount || 0)
  const paid = Number(item.paidAmount || item.paid_amount || 0)
  return Math.max(0, deal - paid).toFixed(2)
}
```
