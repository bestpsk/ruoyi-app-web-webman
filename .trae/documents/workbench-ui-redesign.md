# 工作台页面 UI 优化计划

## 设计参考分析
根据用户提供的设计图，主要特点：
1. **图标风格**：圆形蓝色/绿色背景 + 白色单色线条图标，简洁统一
2. **布局风格**：卡片式设计，圆角，轻微阴影
3. **配色方案**：主色调为蓝色(#3c96f3)，辅以绿色等
4. **常用功能区域**：4列网格，每个图标带文字说明

## 实施步骤

### 1. 更新工作台页面 (work/index.vue)
- 移除 emoji 图标，改用 CSS 绘制的简洁单色图标或 Unicode 符号
- 采用圆形蓝色背景 + 白色图标的样式
- 保持卡片式布局和圆角设计
- 调整配色为简洁的单色系

### 2. 更新我的页面 (mine/index.vue)
- 同样采用简洁的单色图标风格
- 保持与工作台一致的设计语言
- 优化头部区域和菜单列表样式

### 3. 具体设计方案

#### 工作台图标方案（圆形蓝底白字）：
| 模块 | 图标符号 | 背景色 |
|------|----------|--------|
| 用户管理 | 👤 或 U | #3c96f3 |
| 角色管理 | 👥 或 R | #3c96f3 |
| 菜单管理 | ☰ 或 M | #3c96f3 |
| 部门管理 | 🏢 或 D | #3c96f3 |
| 岗位管理 | 💼 或 P | #3c96f3 |
| 字典管理 | 📖 或 B | #3c96f3 |
| 参数设置 | ⚙️ 或 S | #3c96f3 |
| 通知公告 | 🔔 或 N | #3c96f3 |

#### 样式调整：
- 图标容器：圆形 (50px)，统一蓝色背景 (#3c96f3)
- 图标内容：白色，简洁的字母或符号
- 整体风格：扁平化、简洁、专业
