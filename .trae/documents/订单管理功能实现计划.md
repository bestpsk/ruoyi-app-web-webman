# 订单管理功能实现计划

## 需求概述
在"业务管理"菜单下新增一个子菜单"订单管理"，展示所有订单数据列表，支持查询条件筛选，并提供企业审核和财务审核的开关功能。同时显示实付金额和欠款金额。

## 现有资源分析

### 后端已有功能
- **控制器**: `webman/app/controller/business/BizSalesOrderController.php`
  - `list()` - 订单列表查询
  - `enterpriseAudit()` - 企业审核
  - `financeAudit()` - 财务审核
- **服务**: `webman/app/service/BizSalesOrderService.php`
  - 支持按客户、订单号、企业、门店、订单状态、审核状态等条件筛选
- **模型**: `webman/app/model/BizSalesOrder.php`
  - 包含 `enterprise_audit_status`、`finance_audit_status` 等字段
  - **需要添加**: `paid_amount`（实付金额）、`owed_amount`（欠款金额）
- **路由**: 已配置在 `webman/config/route.php`

### 前端已有资源
- **API**: `front/src/api/business/salesOrder.js`
  - `listSalesOrder()` - 查询订单列表
  - `enterpriseAudit()` - 企业审核
  - `financeAudit()` - 财务审核
- **字典**: `biz_order_status` - 订单状态字典已存在

**操作**: 在 `biz_sales_order` 表添加两个字段

- `owed_amount` - 欠款金额（decimal(12,2)，默认0.00）

## 技术要点

### 企业审核/财务审核开关实现
使用 `el-switch` 组件，结合权限控制：
```vue
<el-switch 
  v-model="scope.row.enterpriseAuditStatus" 
  active-value="1" 
  inactive-value="0"
  :disabled="scope.row.enterpriseAuditStatus === '1'"
  @change="handleEnterpriseAudit(scope.row)"
  v-hasPermi="['business:order:enterpriseAudit']"
/>
```

### 查询参数映射
| 前端参数 | 后端参数 |
|---------|---------|
| orderNo | order_no |
| customerName | customer_name |
| enterpriseId | enterprise_id |
| storeId | store_id |
| orderStatus | order_status |
| enterpriseAuditStatus | enterprise_audit_status |
| financeAuditStatus | finance_audit_status |
| dateRange | start_date, end_date |

### 金额字段说明
| 字段 | 说明 |
|-----|------|
| total_amount | 方案总金额 |
| deal_amount | 成交总金额 |
| paid_amount | 实付金额（新增） |
| owed_amount | 欠款金额（新增，= 成交金额 - 实付金额） |
