# 清除进销存测试数据计划（PHP脚本执行）

## 需要清除的数据表

| 表名 | 说明 | 清除方式 |
|------|------|----------|
| biz_stock_in_item | 入库单明细 | DELETE |
| biz_stock_in | 入库单主表 | DELETE |
| biz_stock_out_item | 出库单明细 | DELETE |
| biz_stock_out | 出库单主表 | DELETE |
| biz_stock_check_item | 库存盘点明细 | DELETE |
| biz_stock_check | 库存盘点主表 | DELETE |
| biz_inventory | 库存表 | UPDATE quantity=0, warn_qty=0 |

## 实施方案

创建 PHP 脚本 `webman/clear_wms_data.php`，使用 webman 的数据库连接执行清除操作：

```php
<?php
require_once __DIR__ . '/vendor/autoload.php';
require_once __DIR__ . '/support/bootstrap.php';

use support\Db;

echo "开始清除进销存数据...\n";

// 清除入库单明细
$count = Db::table('biz_stock_in_item')->delete();
echo "清除入库单明细: {$count} 条\n";

// 清除入库单主表
$count = Db::table('biz_stock_in')->delete();
echo "清除入库单主表: {$count} 条\n";

// 清除出库单明细
$count = Db::table('biz_stock_out_item')->delete();
echo "清除出库单明细: {$count} 条\n";

// 清除出库单主表
$count = Db::table('biz_stock_out')->delete();
echo "清除出库单主表: {$count} 条\n";

// 清除库存盘点明细
$count = Db::table('biz_stock_check_item')->delete();
echo "清除库存盘点明细: {$count} 条\n";

// 清除库存盘点主表
$count = Db::table('biz_stock_check')->delete();
echo "清除库存盘点主表: {$count} 条\n";

// 重置库存数量
$count = Db::table('biz_inventory')->update(['quantity' => 0, 'warn_qty' => 0]);
echo "重置库存数量: {$count} 条\n";

echo "清除完成!\n";
```

## 执行方式

```bash
cd d:\fuchenpro\webman

```
