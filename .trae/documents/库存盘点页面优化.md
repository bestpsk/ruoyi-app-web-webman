# 库存盘点页面宽度优化计划

## 需求分析

**当前问题：**
1. 表格没有占满整个内容区（右侧有空白）
2. 列宽使用固定像素值，无法自适应内容区宽度

**目标：**
- 表格宽度 100% 占满内容区
- 各列根据内容自适应分配剩余空间

---

## 实施步骤

### 步骤1：主表格添加 100% 宽度
**文件：** [stockCheck/index.vue](file:///d:/fuchenpro/front/src/views/wms/stockCheck/index.vue) 第31行

```html
<!-- 修改前 -->
<el-table v-loading="loading" :data="stockCheckList" @selection-change="handleSelectionChange">

<!-- 修改后 -->
<el-table v-loading="loading" :data="stockCheckList" @selection-change="handleSelectionChange" style="width: 100%">
```

### 步骤2：列表格列宽改为 min-width 自适应

| 列 | 修改前 | 修改后 |
|---|-------|-------|
| 盘点单号 | width="160" | **min-width="140"** |
| 盘点日期 | width="110" | **min-width="100"** |
| 盘点总数量 | width="110" | **min-width="110"** |
| 差异数量 | width="100" | **min-width="100"** |
| 操作人 | width="100" | **min-width="90"** |
| 状态 | width="90" | **min-width="80"** |
| 创建时间 | width="160" | **min-width="150"** |
| 操作 | width="200" | **min-width="200"** |

**说明：**
- 使用 `min-width` 设置最小宽度
- 当表格宽度 > 所有列 min-width 之和时，各列自动平分剩余空间
- 选择框列保持 `width="55"` 固定
