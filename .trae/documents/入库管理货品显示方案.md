# 入库管理列表 — 货品信息显示方案

## 常见入库管理列表设计

### 主流 WMS/ERP 系统的做法

| 系统 | 显示方式 | 说明 |
|------|----------|------|
| 用友/金蝶 | 只显示主信息 | 点击详情查看明细 |
| 管家婆 | 主信息 + 第一个货品 | 多货品显示"等N项" |
| 旺店通 | 展开行 | 点击展开查看明细 |
| 自定义系统 | 标签展示 | 用标签显示货品名称 |

**结论**：大多数系统确实只显示主信息，但为了提升用户体验，可以增加快捷查看方式。

## 推荐方案

**方案：货品名称列 + 展开行（可点击查看完整明细）**

### 效果预览

```
   入库单号    货品名称              供货商      总数量       总金额   状态   操作
▶ RK001      GCS-p7 等2项         南京伊美荟  10盒(100支)  25800   待确认  查看 修改 删除
▶ RK002      测试货品             供货商A     5个          500     已确认  查看
▼ RK003      GCS-p7               南京伊美荟  20盒(200支)  51600   已确认  查看
  ┌────────────────────────────────────────────────────────────────────┐
  │ 货品名称   规格   换算      数量        单价    金额              │
  │ GCS-p7    支    1盒=10支  20盒(200支) 258/支  51600             │
  └────────────────────────────────────────────────────────────────────┘
```

### 功能说明

1. **货品名称列**：
   - 显示第一个货品名称
   - 如果有多个货品，显示"等N项"
   - 最多显示2个标签，超出显示"+N"

2. **展开行**：
   - 点击左侧展开按钮可查看完整明细
   - 展开后显示该入库单的所有货品明细

3. **供货商列**：
   - 显示第一个货品的供货商
