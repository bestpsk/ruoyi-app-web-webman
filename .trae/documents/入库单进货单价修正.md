# 查看入库单 — 进货单价数字修正

## 问题

查看入库单进货单价显示 `680/盒`，但应该显示 `6800/盒`。

## 根因

[BizStockInService.php#L111-L113](file:///d:\fuchenpro\webman\app\service\BizStockInService.php#L111-L113)：

```php
if ($unitType === '1' && $packQty > 1) {
    $item['quantity'] = intval($item['quantity']) * $packQty;     // 数量 × 换算
    if (isset($item['_main_price']) && $item['_main_price'] > 0) {
        $item['purchase_price'] = bcdiv($item['_main_price'], $packQty, 4); // 单价 ÷ 换算 ❌
    }
}
```

当选择**主单位(整)**入库时，后端将 `purchasePrice` 除以了 `packQty`：
- 用户输入：数量=1盒，单价=6800元/盒
- 数据库存储：`quantity=100`(最小单位)，`purchasePrice=680`(100支时的单价)
- 查看时应还原：`6800/盒`（主单位单价）

## 数据库实际存储值

| 字段 | 用户输入 | 数据库存储 |
|------|---------|-----------|
| quantity | 1 | 100 (=1×10) |
| original_quantity | 1 | 1 |
| purchasePrice | 6800 | **680** (=6800÷10) |

## 修复方案

修改 `formatItemPrice` 方法，根据 unitType 还原正确的单价：

```javascript
function formatItemPrice(item) {
  const unitLabel = getUnitLabel(item.unit)
  const specLabel = getSpecLabel(item.spec)
  const packQty = item.packQty || 1
  
  if (item.unitType === '1') {
    // 主单位(整)：数据库存的是副单位单价，需要 × packQty 还原
    let price = item.purchasePrice || 0
    if (packQty > 1) {
      price = parseFloat((price * packQty).toFixed(2))
    }
    return price + '/' + (unitLabel || '')
  } else {
    // 副单位(拆)：直接使用数据库中的单价
    return item.purchasePrice + '/' + (specLabel || '')
  }
}
```
