# 订单详情页：去除重复信息并优化布局

## 问题分析

### 当前状态（从截图可见）
```
┌─────────────────────────────────────┐
│ ① 额电饭锅电饭锅              已成交 │  ← 第1行：项目名称
│                                     │
│ 品项名 额电饭锅电饭锅          次数 10次│  ← 第2行：❌ 重复！
│ 方案价 ¥3980.00              单价 ¥398│  ← 第3行
│ 成交价 ¥3980.00              实付 ¥3980│  ← 第4行
└─────────────────────────────────────┘
```

**核心问题**: 
- ❌ **信息重复**：第1行已显示"额电饭锅电饭锅"，第2行又显示"品项名: 额电饭锅电饭锅"
- ❌ **次数位置不合理**：次数应该紧跟项目名称

---

## 优化目标

```
┌─────────────────────────────────────┐
│ ① 额电饭锅电饭锅          10次   已成交│  ← 标题+次数靠右
│                                     │
│ 方案价 ¥3980.00              单价 ¥398│  ← 价格信息
│ 成交价 ¥3980.00              实付 ¥3980│  ← 金额汇总
└─────────────────────────────────────┘
```

**改进点**:
- ✅ 删除"品项名"行（消除重复）
- ✅ 将"N次"移到项目名称后面，靠右对齐
- ✅ 信息更紧凑，无冗余

## 技术实现要点

### Flexbox 两端对齐
```scss
.item-header {
  display: flex;
  justify-content: space-between;  // 左右分布
}

.item-left { flex: 1; }           // 左侧占据剩余空间
.item-right { margin-left: auto; } // 右侧自动靠右
```

### 布局原理
```
┌── item-header (flex container) ──────────────────────┐
│                                                        │
│  ── item-left (flex: 1) ──┐  ┌─ item-right (auto) ──│
│  │                         │  │                     │   │
│  │ ① 名称                  │  │ 10次    [已成交]     │   │
│  └─────────────────────────┘  └─────────────────────┘   │
│         ↑ 自动扩展               ↑ 靠右对齐            │
└────────────────────────────────────────────────────────┘
```
