# 出货时支持自由选择货品方案

## 业务场景分析

### 当前问题
1. 新建方案时可能没有填写配赠明细（企业已付款、折扣谈好，但不着急拿货）
2. 出货时只能从配赠明细中选择，没有配赠明细则无法出货
3. 剩余金额存在但无法使用

### 用户需求
- 方案审核通过后，即使没有配赠明细也能出货
- 出货时可自由选择货品库中的任何货品
- 价格可按方案折扣或自定义

---

## 解决方案：混合模式（推荐）

### 设计思路

出货明细支持两种来源：
1. **配赠明细**：方案预设的配赠商品（如有）
2. **自由选择**：从货品库中自由添加其他货品

### UI设计

```
┌─────────────────────────────────────────────────────────────┐
│  关联方案: 龚田诗 30%方案    企业名称: 龚田诗                │
│  收货人: [汪志]   收货电话: [15888888888]                   │
├─────────────────────────────────────────────────────────────┤
│  出货明细                                                    │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 货品名称 | 供货商 | 单位类型 | 换算 | 数量 | 单价 | ... ││
│  │ 测试1    | 南京伊 | 主单位整 | ...  | 10  | 6800 | ... ││  ← 配赠明细（自动加载）
│  └─────────────────────────────────────────────────────────┘│
│  [+ 添加其他货品]                                            │  ← 新增按钮
└─────────────────────────────────────────────────────────────┘
```

点击"添加其他货品"后：
- 弹出货品选择器（或直接在表格中新增一行可选择货品的行）
- 选择货品后自动填充信息
- 折扣单价默认为原价，可手动修改

## 实施步骤

### 步骤1：修改出货弹窗UI

**文件**: `front/src/views/business/planList/index.vue`

修改内容：
1. 在出货明细表格下方添加"添加其他货品"按钮
2. 点击后新增一行，货品名称列变为下拉选择框
3. 选择货品后自动填充供货商、单位类型、换算等信息

### 步骤2：添加货品选择逻辑

新增函数：
- `addCustomProduct()` - 添加空白行用于选择货品
- `onShipmentProductSelect(index, productId)` - 处理货品选择
- `searchShipmentProduct(query)` - 搜索货品

### 步骤3：调整提交逻辑

修改 `submitShipmentForm()`：
- 区分配赠明细项（有 planItemId）和自定义项（无 planItemId）
- 自定义项需要包含完整的货品信息

### 步骤4：后端适配（如需要）

检查 `addShipment` API 是否支持接收自定义货品数据。

## 数据结构说明

### 配赠明细项（原有）
```javascript
{
  planItemId: 123,        // 配赠明细ID（必填）
  productId: 1,
  productName: '测试1',
  // ... 其他字段
}
```

### 自定义新增项（新）
```javascript
{
  planItemId: undefined,  // 无配赠明细ID
  productId: 5,
  productName: '新品',
  supplierId: 2,
  supplierName: '供应商B',
  unitType: '1',
  packQty: 10,
  quantity: 5,
  spec: '盒',
  salePrice: 6800,
  discountPrice: 6000,    // 可修改的折扣价
  amount: 30000,
  // ...
}
```

## 预期效果

1. 有配赠明细的方案：正常显示配赠明细，也可额外添加其他货品
2. 无配赠明细的方案：显示空列表 + "添加其他货品"按钮
3. 用户可从货品库自由选择货品进行出货
4. 支持多次出货，直到剩余金额用完
