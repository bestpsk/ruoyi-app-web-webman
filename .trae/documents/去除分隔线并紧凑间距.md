# 订单详情页去除分隔线并紧凑行间距

## 需求分析

根据用户选中的HTML元素和截图反馈，需要优化以下两处：

### 位置1：金额行上方虚线 ⭐⭐
**选中元素**: `<uni-view class="info-row amount-row">`
**当前代码** (detail.vue 第263-267行):
```scss
&.amount-row {
  margin-top: 8rpx;
  padding-top: 16rpx;
  border-top: 1rpx dashed #E5E6EB;  // ← 虚线需删除
}
```

**问题**: 金额行上方有虚线分隔符，视觉上不够简洁

### 位置2：订单项目汇总行上方实线 ⭐⭐
**选中元素**: `<uni-view class="info-line summary-line">`
**当前代码** (detail.vue 第437-441行):
```scss
&.summary-line {
  margin-top: 8rpx;
  padding-top: 12rpx;
  border-top: 1rpx solid #F2F3F5;  // ← 实线需删除
}
```

**问题**: 成交/实付行上方有实线分隔符，增加视觉噪音

### 整体间距问题 ⭐
**当前设置**:
- `.info-body { gap: 16rpx }` → 行间距偏大
- 各处 `margin-top` / `padding-top` 累加导致松散

---

## 优化方案

### 核心原则
✅ **去除装饰性分隔线** - 虚线和实线都删除  
✅ **紧凑化间距** - 减小gap、margin、padding  
✅ **保持层次感** - 通过字重和颜色区分，无需线条  

---

## 具体修改步骤

### 步骤1：修改金额行样式（去除虚线+减小间距）

**文件**: `d:\fuchenpro\AppV3\src\pages\business\order\detail.vue`
**位置**: 第263-267行

**修改前**:
```scss
&.amount-row {
  margin-top: 8rpx;
  padding-top: 16rpx;
  border-top: 1rpx dashed #E5E6EB;  // 虚线
}
```


**效果**: 
- ❌ 移除虚线分隔符

## 技术实现要点

### 删除的样式属性
```scss
// 金额行 - 删除
border-top: 1rpx dashed #E5E6EB;

// 汇总行 - 删除  
border-top: 1rpx solid #F2F3F5;
```

### 调整的间距值
| 属性 | 修改前 | 修改后 | 变化量 |
|------|--------|--------|--------|
| `.info-body` gap | 16rpx | **10rpx** | ↓ 37.5% |
| `.amount-row` margin-top | 8rpx | **4rpx** | ↓ 50% |
| `.amount-row` padding-top | 16rpx | **8rpx** | ↓ 50% |
| `.summary-line` margin-top | 8rpx | **4rpx** | ↓ 50% |
| `.summary-line` padding-top | 12rpx | **6rpx** | ↓ 50% |

### 保持的设计原则
✅ 通过**字重**(font-weight)区分重要性  
✅ 通过**颜色**(color)区分信息类型  
✅ 通过**字号**(font-size)建立层级  
❌ 不再依赖**分割线**来分组
