# 新增企业表单美化计划

## 当前问题
1. `labelPosition="top"` 导致每个字段独占一行，标签在输入框上方，非常松散
2. 输入框没有图标，视觉单调
3. 表单项之间间距过大
4. 整体不够紧凑美观

## 修改目标
- 去掉 u-form-item 的 label，改为**输入框内部前缀插槽 + 图标**
- 每个输入框左侧显示图标+简短标签文字
- 紧凑布局，减少间距
- 门店数量和年业绩保持并排一行
- 整体风格更现代、更紧凑

## 设计方案

### 具体改动

#### 1. 模板部分 - 改为自定义输入框样式
- 移除所有 `<u-form-item label=...>` 的 label 属性
- 改为使用自定义的 `.form-field` 容器包裹每个字段
- 每个 field 内部：`<view class="field-label"><u-icon><text></text></view>` + `<input class="field-input">`
- 使用原生 `<input>` 替代 `<u-input>`，方便控制前缀插槽
- 选择器类型字段右侧显示箭头图标

#### 2. 字段配置表

| 字段 | 图标 | 前缀文字 | 类型 |
|------|------|----------|------|
| 企业名称 | home-fill | 企业名称 | text |
| 老板姓名 | account | 老板姓名 | text |
| 联系电话 | phone | 电话 | number |
| 企业类型 | shop | 类型 | picker(右箭头) |
| 地址 | map | 地址 | textarea |
| 门店数量 | grid | 门店数 | number(半宽) |
| 年业绩 | red-packet-fill | 业绩 | digit(半宽) |
| 企业级别 | star-fill | 级别 | picker(右箭头) |
| 服务人 | man-add | 服务人 | text |
| 状态 | checkmark-circle | 状态 | radio |
| 备注 | edit-pen | 备注 | textarea |

#### 3. 样式调整
- `.form-section`: padding 减小到 `24rpx`
- `.form-field`: 每个字段的容器，间距 `20rpx`
- `.field-input-box`: 白色背景圆角框，flex 布局
- `.field-prefix`: 左侧图标+文字区域，固定宽度，灰色文字
- `.field-input`: flex:1，无边框
- 半宽字段（门店/业绩）：`.half-width` 类，flex:0.48

#### 4. 验证规则保留
- 保留 rules 对象用于提交前校验
- 必填项在 placeholder 中加 * 号提示
