# 入库/出库管理列表 - 统一大单位（小单位）显示格式

## 显示格式规范

**统一格式**：`{大单位数量}{大单位名}（{小单位数量}{小单位名}）`

| 场景 | 显示效果 | 说明 |
|------|----------|------|
| 有换算关系 | 1套（10支） | 大单位+小单位 |
| 有换算关系（非整数） | 0.5套（5支） | 支持小数 |
| 无换算关系 | 1件 | 没有小单位就不显示括号 |

## 数据计算逻辑

- `totalQuantity`（数据库存储）= 最小单位数量
- 大单位数量 = `totalQuantity / packQty`
- 小单位数量 = `totalQuantity`

**示例**：
- 入库2套，1套=10支 → totalQuantity=20, packQty=10 → 显示：2套（20支）
- 出库5支，1套=10支 → totalQuantity=5, packQty=10 → 显示：0.5套（5支）
- 入库3件，无换算 → totalQuantity=3, packQty=1 → 显示：3件

## 实施步骤

### 步骤1：修改入库后端 - BizStockInService.php
在 `selectStockInList` 方法中添加 display 字段：

```php
foreach ($list->items() as $stockIn) {
    $firstItem = BizStockInItem::where('stock_in_id', $stockIn->stock_in_id)->first();
    if ($firstItem) {
        $stockIn->display_pack_qty = $firstItem->pack_qty ?? 1;
        $stockIn->display_unit = $firstItem->unit;
        $stockIn->display_spec = $firstItem->spec;
    }
}
```

注意：不需要 display_unit_type 和 display_original_qty，因为统一按大单位（小单位）格式显示，只需 pack_qty、unit、spec 即可计算。

### 步骤2：修改入库前端 - stockIn/index.vue
总数量列改为带单位的模板，添加格式化方法：

```vue
<el-table-column label="总数量" min-width="120" align="center">
  <template #default="scope">
    {{ formatQuantity(scope.row) }}
  </template>
</el-table-column>
```

添加格式化方法：
```javascript
function formatQuantity(row) {
  const packQty = row.display_pack_qty || 1
  const unitLabel = getUnitLabel(row.display_unit)
  const specLabel = getSpecLabel(row.display_spec)
  const totalQty = row.totalQuantity || 0

  if (packQty > 1 && specLabel) {
    const mainQty = totalQty / packQty
    const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
    return mainQtyStr + unitLabel + '（' + totalQty + specLabel + '）'
  } else if (unitLabel) {
    return totalQty + unitLabel
  } else {
    return totalQty
  }
}
```

### 步骤3：修改出库前端 - stockOut/index.vue
同样优化总数量列，使用统一的格式化方法：

```vue
<el-table-column label="总数量" min-width="160" align="center">
  <template #default="scope">
    {{ formatQuantity(scope.row) }}
  </template>
</el-table-column>
```

添加相同的 formatQuantity 方法。

### 步骤4：出库后端无需修改
BizStockOutService.php 已有 display_pack_qty、display_unit、display_spec 字段。
