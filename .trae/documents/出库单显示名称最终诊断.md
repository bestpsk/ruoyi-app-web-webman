## 备选快速修复方案

### 方案：直接更新数据库旧数据（临时解决方案）

如果后端代码确实无法生效，可以先修复现有数据：

```sql
-- 更新所有存储为数字的企业名称
UPDATE biz_stock_out so
JOIN biz_enterprise e ON CAST(so.enterprise_id AS UNSIGNED) = e.enterprise_id
SET so.enterprise_name = e.enterprise_name
WHERE CAST(so.enterprise_name AS SIGNED) = so.enterprise_id;

-- 更新所有存储为数字的员工名称
UPDATE biz_stock_out so
JOIN sys_user u ON CAST(so.contact_employee_id AS UNSIGNED) = u.user_id
SET so.contact_employee_name = COALESCE(u.real_name, u.user_name)
WHERE CAST(so.contact_employee_name AS SIGNED) = so.contact_employee_id;
```
