# 入库单操作人保存与打印签名优化计划

## 需求分析

### 问题1：新增入库单时操作人没有被正确保存
**现象：**
- 新增入库单后，操作人显示为 `-`
- 打印预览中"制单人"为空

**原因分析：**

查看后端控制器 [BizStockInController.php](file:///d:/fuchenpro/webman/app/controller/wms/BizStockInController.php) 第35行：

```php
$data['operator_name'] = $request->loginUser->user->real_name ?? '';
```

对比其他控制器的写法（如 [BizStockOutController.php](file:///d:/fuchenpro/webman/app/controller/wms/BizStockOutController.php) 第33行）：

```php
$data['create_by'] = $request->loginUser->user->real_name ?? $request->loginUser->user->user_name ?? '';
```

**问题：**
- 当前代码只使用 `real_name`，没有 `user_name` 作为回退
- 如果 `real_name` 为空字符串（而非 null），`??` 运算符不会触发回退

**解决方案：**
- 参照其他控制器的写法，增加 `user_name` 回退
- 使用 `trim()` 确保空字符串也能正确处理

---

### 问题2：打印预览签名区域横线不统一
**当前状态：**

| 字段 | 当前实现 | 横线 |
|-----|---------|------|
| 制单人 | `{{ form.operatorName }}` | **无横线** |
| 经手人 | `<span class="underline">` | 60px短横线 |
| 送货签字 | `<span class="underline-long">` | 150px长横线 |

**用户期望：**
- 统一签名区域的横线样式

---

## 实施步骤

### 步骤1：修复后端操作人保存逻辑
修改 [BizStockInController.php](file:///d:/fuchenpro/webman/app/controller/wms/BizStockInController.php) 第35行：

```php
// 当前代码
$data['operator_name'] = $request->loginUser->user->real_name ?? '';

// 修改为 - 与其他控制器保持一致，增加user_name回退
$realName = trim($request->loginUser->user->real_name ?? '');
$userName = trim($request->loginUser->user->user_name ?? '');
$data['operator_name'] = $realName ?: $userName;
```

---

### 步骤2：统一打印预览签名横线
修改 [index.vue](file:///d:/fuchenpro/front/src/views/wms/stockIn/index.vue) 第233-242行：

**当前代码：**
```html
<div class="print-footer">
  <div class="footer-row">
    <div class="footer-item"><span>制单人：</span>{{ form.operatorName || '' }}</div>
    <div class="footer-item"><span>经手人：</span><span class="underline"></span></div>
    <div class="footer-item"><span>日期：</span>{{ form.stockInDate }}</div>
  </div>
  <div class="footer-row sign-row">
    <div class="footer-item"><span>送货签字：</span><span class="underline-long"></span></div>
  </div>
</div>
```

**修改为（统一横线样式）：**
```html
<div class="print-footer">
  <div class="footer-row">
    <div class="footer-item">
      <span>制单人：</span><span class="sign-line">{{ form.operatorName || '' }}</span>
    </div>
    <div class="footer-item">
      <span>经手人：</span><span class="sign-line"></span>
    </div>
    <div class="footer-item">
      <span>日期：</span>{{ form.stockInDate }}
    </div>
  </div>
  <div class="footer-row sign-row">
    <div class="footer-item">
      <span>送货签字：</span><span class="sign-line-long"></span>
    </div>
  </div>
</div>
```

## 验证方式

### 操作人验证：
1. 使用已填写姓名的账号登录
2. 新增一个入库单并保存
3. 检查列表中"操作人"列是否显示用户姓名
4. 点击查看确认详情页操作人正确

### 打印签名验证：
1. 点击打印按钮预览
2. 确认签名区域：
   - ✅ 制单人后面有横线，显示操作人姓名
   - ✅ 经手人后面有相同宽度的空白横线
   - ✅ 送货签字后有较长的空白横线
   - ✅ 所有横线风格统一
