# 多货品入库单 — 总数量最终方案

## 大单位 vs 小单位对比

| 维度 | 大单位（整） | 小单位（拆） |
|------|-------------|-------------|
| **直观性** | ✅ `10盒` 简洁 | ❌ `100支` 数字大 |
| **业务习惯** | ✅ 采购/销售按箱报 | ❌ 仓库按支点数 |
| **适用场景** | ✅ 管理/财务报表 | ✅ 库存/盘点 |
| **多货品问题** | ❌ 不同单位无法汇总 | ⚠️ 统一为最小单位可汇总 |

## 推荐结论

**入库管理列表 → 显示大单位（整）**

理由：
1. 入库单是按"整"来操作的（如：进了10盒）
2. 符合采购/管理视角的阅读习惯
3. 数字简洁易读

**对于多货品不同规格的情况**：

| 场景 | 推荐显示 |
|------|----------|
| 单个货品 | `10盒`（大单位） |
| 多个同规格货品 | `20盒`（汇总大单位） |
| 多个**不同规格**货品 | 只显示**种类数**，不显示数量 |

因为不同规格无法统一单位，强行显示反而误导。

### 最终效果

```
入库单号    总数量              总金额     操作
RK001      10盒                25800.00   查看    ← 单个货品
RK002      2种                 51800.00   查看    ← 不同规格
RK003      3盒                  7740.00   查看    ← 同规格汇总
```

## 实施步骤

修改 `formatQuantity` 方法：

```javascript
function formatQuantity(row) {
  const items = row.items || []
  
  if (items.length === 0) {
    return row.totalQuantity || 0
  }
  
  // 单个货品：显示大单位
  if (items.length === 1) {
    const item = items[0]
    const packQty = row.display_pack_qty || item.packQty || 1
    const unitLabel = getUnitLabel(row.display_unit || item.unit)
    const totalQty = row.totalQuantity || 0
    if (packQty > 1 && unitLabel) {
      const mainQty = totalQty / packQty
      const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
      return mainQtyStr + unitLabel
    } else if (unitLabel) {
      return totalQty + unitLabel
    } else {
      return totalQty
    }
  }
  
  // 多个货品：检查是否同规格（同 unit）
  const firstUnit = items[0].unit
  const isSameUnit = items.every(item => item.unit === firstUnit)
  
  if (isSameUnit) {
    // 同规格：汇总显示大单位
    const packQty = items[0].packQty || 1
    const unitLabel = getUnitLabel(firstUnit)
    const totalQty = row.totalQuantity || 0
    if (packQty > 1 && unitLabel) {
      const mainQty = totalQty / packQty
      const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
      return mainQtyStr + unitLabel
    } else {
      return totalQty + (unitLabel || '')
    }
  } else {
    // 不同规格：只显示种类数
    return items.length + '种'
  }
}
```
