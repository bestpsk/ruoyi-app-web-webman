### 问题2：供货商显示"未知供货商"
**根本原因**：
- `BizProduct` 模型只有 `supplier_id` 字段，没有 `supplierName` 字段

- 前端代码 `product.supplierName` 取不到值，所以显示"未知供货商"

**代码位置**：
- 后端：[BizProductService.php:38-53](file:///d:/fuchenpro/webman/app/service/BizProductService.php#L38-L53) 的 `searchProduct` 方法

### 后端修改（BizProductService.php）

修改 `searchProduct` 方法：
```php
public function searchProduct($keyword = '')
{
    $query = BizProduct::query()->where('status', '0');
    if (!empty($keyword)) {
        $query->where(function ($q) use ($keyword) {
            $q->where('product_name', 'like', '%' . $keyword . '%')
              ->orWhere('product_code', 'like', '%' . $keyword . '%');
        });
    }
    $products = $query->with('supplier')->orderBy('product_id', 'desc')->limit(50)->get();
    foreach ($products as $product) {
        $inventory = BizInventory::where('product_id', $product->product_id)->first();
        $product->inventory_qty = $inventory ? $inventory->quantity : 0;
        $product->supplier_name = $product->supplier ? $product->supplier->supplier_name : null;
    }
    return $products;
}
```
