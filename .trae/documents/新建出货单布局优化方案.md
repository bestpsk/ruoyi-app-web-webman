# 新建出货单弹窗布局优化方案

## 问题分析

当前布局问题：
1. **总金额/剩余金额行**：使用 `flex + space-between`，与表格不对齐
2. **备注区域**：`label-width="100px"` 与上方表单不一致
3. **整体视觉**：各元素间距不统一，看起来凌乱

## 当前代码结构（问题区域）

```html
<!-- 添加其他货品按钮 -->
<div style="margin-top: 10px">
  <el-button type="primary" link icon="Plus">添加其他货品</el-button>
</div>

<!-- 金额汇总：左右分布，与表格不统一 -->
<div style="margin-top: 10px; display: flex; justify-content: space-between">
  <span>总金额: xxx</span>
  <span>方案剩余金额: xxx</span>
</div>

<!-- 备注：独立一行，label-width与上方不一致 -->
<el-row style="margin-top: 10px">
  <el-col :span="24">
    <el-form-item label="备注" prop="remark">...</el-form-item>
  </el-col>
</el-row>
```

## 优化方案

### 布局设计

```
┌─────────────────────────────────────────────────────────────┐
│  出货明细                                                    │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ 表格内容...                                              ││
│  └─────────────────────────────────────────────────────────┘│
│  [+ 添加其他货品]                                            │
├─────────────────────────────────────────────────────────────┤
│  总金额: 680.00              方案剩余金额: 100000.00          │  ← 右对齐表格边框
├─────────────────────────────────────────────────────────────┤
│  备注  [________________________________________]           │  ← 左对齐表格边框
└─────────────────────────────────────────────────────────────┘
```

### 具体修改

**文件**: `front/src/views/business/planList/index.vue`

将第326-339行的代码替换为：

```html
<div style="margin-top: 12px; padding: 0 2px">
  <el-button type="primary" link icon="Plus" @click="addCustomProduct" style="margin-bottom: 8px">添加其他货品</el-button>
</div>

<div style="margin-top: 16px; display: flex; justify-content: flex-end; gap: 40px; padding-right: 4px">
  <span style="font-weight: 500">总金额: <strong>{{ shipmentTotalAmount }}</strong></span>
  <span style="color: #909399">方案剩余金额: {{ shipmentForm.remainingAmount }}</span>
</div>

<div style="margin-top: 16px">
  <el-form-item label="备注" prop="remark" label-width="52px">
    <el-input v-model="shipmentForm.remark" type="textarea" placeholder="请输入备注" :rows="3" />
  </el-form-item>
</div>
```

## 修改要点

| 元素 | 原来 | 优化后 |
|------|------|--------|
| 添加按钮 | 独立div | 统一padding |
| 金额行 | space-between | **右对齐** + 加粗总金额 |
| 金额间距 | 无 | `gap: 40px` |
| 备注 | label-width=100px | **label-width=52px** |
| 备注textarea | 默认高度 | **rows=3** |
| 各元素间距 | 不统一 | **统一16px** |

## 预期效果

1. 金额信息右对齐，与表格右侧对齐
2. 备注左对齐，与表格左侧对齐
3. 间距统一，视觉整洁
4. 总金额加粗突出显示
