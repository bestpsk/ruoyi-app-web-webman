# 角色管理权限列表优化方案

## 问题分析

当前角色管理的菜单权限树形结构：
```
系统管理
├── 用户管理
│   ├── 用户查询 (按钮)
│   ├── 用户新增 (按钮)
│   ├── 用户修改 (按钮)
│   └── 用户删除 (按钮)
├── 角色管理
│   ├── 角色查询 (按钮)
│   └── ...
```

**问题**：
1. 弹窗宽度只有500px，权限列表太长
2. 按钮权限作为子节点显示，增加了树的深度
3. 页面需要频繁滚动才能看到所有权限

## 优化方案

### 方案：按钮权限显示在菜单同一行

将按钮权限从树形子节点改为显示在菜单节点的右侧：

```
系统管理
├── 用户管理  [查询] [新增] [修改] [删除] [导出]
├── 角色管理  [查询] [新增] [修改] [删除]
├── 菜单管理  [查询] [新增] [修改] [删除]
```

### 实施步骤

**文件**: `front/src/views/system/role/index.vue`

#### 步骤1：弹窗宽度改为1000px

```html
<!-- 原来 -->
<el-dialog :title="title" v-model="open" width="500px" append-to-body>

<!-- 改为 -->
<el-dialog :title="title" v-model="open" width="1000px" append-to-body>
```

#### 步骤2：修改树形结构显示方式

使用 el-tree 的自定义节点内容，将按钮权限显示在菜单右侧：

```html
<el-tree
  class="tree-border"
  :data="menuOptions"
  show-checkbox
  ref="menuRef"
  node-key="id"
  :check-strictly="!form.menuCheckStrictly"
  empty-text="加载中，请稍候"
  :props="{ label: 'label', children: 'children' }"
>
  <template #default="{ node, data }">
    <span class="custom-tree-node">
      <span>{{ node.label }}</span>
      <span v-if="data.children && data.children.length > 0 && isButtonGroup(data.children)" class="button-permissions">
        <el-checkbox
          v-for="btn in data.children"
          :key="btn.id"
          :label="btn.id"
          :checked="isMenuChecked(btn.id)"
          @change="(val) => handleButtonCheck(btn.id, val)"
        >{{ btn.label }}</el-checkbox>
      </span>
    </span>
  </template>
</el-tree>
```

#### 步骤3：添加辅助函数

```javascript
// 判断子节点是否都是按钮权限（没有children的叶子节点）
function isButtonGroup(children) {
  return children.every(child => !child.children || child.children.length === 0)
}

// 检查菜单是否被选中
function isMenuChecked(menuId) {
  return menuRef.value?.getCheckedKeys().includes(menuId) || false
}

// 处理按钮权限勾选
function handleButtonCheck(menuId, checked) {
  if (checked) {
    menuRef.value.setChecked(menuId, true, false)
  } else {
    menuRef.value.setChecked(menuId, false, false)
  }
}
```

#### 步骤4：添加样式

```css
<style scoped>
.custom-tree-node {
  display: flex;
  align-items: center;
  flex: 1;
  padding-right: 8px;
}
.button-permissions {
  margin-left: 20px;
  display: flex;
  flex-wrap: wrap;
  gap: 8px;
}
.button-permissions .el-checkbox {
  margin-right: 0;
}
</style>
```

## 预期效果

1. 弹窗宽度1000px，显示更宽敞
2. 按钮权限显示在菜单同一行，减少树深度
3. 保持原有功能不变（全选、展开折叠、父子联动等）
4. 后端数据格式不变，只改前端显示方式
