# 打印预览优化计划

## 需求分析

根据用户要求和当前截图，需要修改入库单打印预览：

### 当前问题
1. 列顺序不合理（规格应该在数量后面）
2. 数量带单位显示（应该只显示数字）
3. **表格内容超出纸张宽度**，需要适配三联单

### 目标状态
- 适配三联打印纸（241mm宽针式打印纸或A4 210mm）
- 列顺序：序号 | 货品名称 | 供货商 | 换算 | 数量 | 规格 | 单价 | 金额
- 数量只显示数字
- 规格动态显示货品实际规格

## 实施步骤

### 步骤1：调整容器宽度适配三联单
修改 `.print-container` 样式（第669-678行）：

**当前：** `width: 190mm`  
**修改为：** `width: 210mm` （A4纸宽度，适合大多数打印机）

同时调整 `padding` 从 `10mm` 减少到 `8mm` 以增加可用空间。

### 步骤2：优化表格列宽
修改表头列定义（第203-211行），使用百分比或更小的固定宽度：

| 列名 | 当前宽度 | 优化后 |
|------|---------|--------|
| 序号 | width="45" | width="40" |
| 货品名称 | min-width:120px | width="18%" |
| 供货商 | min-width:90px | width="14%" |
| 换算 | min-width:90px | width="12%" |
| 数量 | min-width:130px | width="10%" |
| 规格 | min-width:55px | width="8%" |
| 单价 | min-width:90px | width="14%" |
| 金额 | min-width:95px | width="14%" |

**关键变更：**
- 规格列从第4位移到第5位（数量后面）
- 移除过大的min-width，改用百分比自适应
- 减小padding节省空间

### 步骤3：调整数据列顺序
修改表格体 `<tbody>` 中的列顺序（第215-224行），与表头一致：

```html
<tr v-for="(item, index) in form.items" :key="index">
  <td align="center">{{ index + 1 }}</td>
  <td>{{ item.productName }}</td>
  <td>{{ item.supplierName || '-' }}</td>
  <td align="center">换算内容</td>
  <td align="center">{{ formatItemQuantity(item) }}</td>  <!-- 只显示数字 -->
  <td align="center">{{ getSpecLabel(item.spec) }}</td>   <!-- 规格移到这里 -->
  <td align="right">{{ formatItemPrice(item) }}</td>
  <td align="right">{{ item.amount }}</td>
</tr>
```

### 步骤4：修改数量格式化函数
修改 `formatItemQuantity` 函数（第374-382行）：

```javascript
function formatItemQuantity(item) {
  return item.originalQuantity || item.quantity || 0
}
```

### 步骤5：调整合计行colspan
更新tfoot中的colspan（第228行），确保合计标签位置正确。

### 步骤6：减小单元格内边距
修改样式（第715-719行），减小padding：

**当前：** `padding: 6px 8px`
**修改为：** `padding: 4px 6px`

### 步骤7：同步更新打印窗口样式
修改 `confirmPrint` 函数中的内联样式（第617-637行），确保打印输出与预览一致。

## 三联单尺寸参考

| 纸张类型 | 尺寸 | 适用场景 |
|---------|------|---------|
| 针式打印纸(三联) | 241mm × 139.7mm | 针式打印机 |
| A4纸 | 210mm × 297mm | 激光/喷墨打印机 |
| A5纸 | 148mm × 210mm | 小型单据 |

**建议采用210mm (A4宽度)**，兼容性最好。

## 验证方式

1. 打开入库单查看/新增页面
2. 点击打印按钮查看预览
3. 确认：
   - ✅ 内容不超出纸张边界
   - ✅ 列顺序正确：序号、货品名称、供货商、换算、数量、规格、单价、金额
   - ✅ 数量只显示数字（如 `1` 而非 `1盒`）
   - ✅ 规格显示货品规格（如 `支`）
   - ✅ 打印输出与预览一致
