# 销售开单表格列修改计划

## 需求概述
在销售开单页面，选择顾客后，对开单表格进行以下修改：
1. 在"单次价"列后面添加两列：**实付金额**（默认等于成交金额，可修改）、**欠款金额**（默认等于0，可修改）
2. 删除表格中的"备注"列
3. 将表格下方的"顾客反馈"改为"备注"

## 涉及文件
- `d:\fuchenpro\front\src\views\business\sales\index.vue`

## 详细修改步骤

### 步骤1：修改表格列定义（第110-119行）

**当前代码：**
```vue
<el-table-column label="单次价" width="100" align="center">
  <template #default="scope">
    <span>{{ scope.row.quantity > 0 && scope.row.isDeal ? (scope.row.dealAmount / scope.row.quantity).toFixed(2) : '0.00' }}</span>
  </template>
</el-table-column>
<el-table-column label="备注" min-width="100">
  <template #default="scope">
    <el-input v-model="scope.row.remark" placeholder="备注" />
  </template>
</el-table-column>
```

**修改为：**
```vue
<el-table-column label="单次价" width="100" align="center">
  <template #default="scope">
    <span>{{ scope.row.quantity > 0 && scope.row.isDeal ? (scope.row.dealAmount / scope.row.quantity).toFixed(2) : '0.00' }}</span>
  </template>
</el-table-column>
<el-table-column label="实付金额" width="140">
  <template #default="scope">
    <el-input-number v-model="scope.row.paidAmount" :min="0" :precision="2" :disabled="!scope.row.isDeal" controls-position="right" style="width: 100%" />
  </template>
</el-table-column>
<el-table-column label="欠款金额" width="140">
  <template #default="scope">
    <el-input-number v-model="scope.row.debtAmount" :min="0" :precision="2" :disabled="!scope.row.isDeal" controls-position="right" style="width: 100%" />
  </template>
</el-table-column>
```

### 步骤3：修改数据初始化（第595-599行）

```javascript
function addOrderItemRow() {
```

**修改为：**

```javascript
function addOrderItemRow() {
  orderItems.value.push({
```

### 步骤4：修改成交状态变更处理函数（第601-607行）

**当前代码：**
```javascript
function handleDealChange(index, val) {
  if (val === 1) {
    orderItems.value[index].dealAmount = orderItems.value[index].planPrice
  } else {
    orderItems.value[index].dealAmount = 0
  }
}
```

**修改为：**
```javascript
function handleDealChange(index, val) {
  if (val === 1) {
    orderItems.value[index].dealAmount = orderItems.value[index].planPrice
    orderItems.value[index].paidAmount = orderItems.value[index].planPrice
    orderItems.value[index].debtAmount = 0
  } else {
    orderItems.value[index].dealAmount = 0
    orderItems.value[index].paidAmount = 0
    orderItems.value[index].debtAmount = 0
  }
}
```

### 步骤5：修改提交订单数据（第609-637行）

在提交订单时，需要将 `customerFeedback` 字段改为 `remark` 字段（因为前端显示已改为"备注"），同时保留 items 中的 paidAmount 和 debtAmount 字段。

**当前代码：**

**修改为：**
```javascript
const data = {
  customerId: currentCustomer.value.customerId,
  customerName: currentCustomer.value.customerName,
  enterpriseId: currentEnterpriseId.value,
  enterpriseName: ent?.enterpriseName,
  storeId: currentStoreId.value,
```

## 修改总结

| 修改项 | 位置 | 说明 |
|--------|------|------|
| 添加实付金额列 | 第110行后 | 新增列，默认值等于成交金额 |
| 添加欠款金额列 | 实付金额列后 | 新增列，默认值等于0 |
| 删除备注列 | 第115-119行 | 移除表格中的备注列 |
| 修改标签文字 | 第127行 | "顾客反馈" → "备注" |
| 数据初始化 | 第595-599行 | 添加 paidAmount, debtAmount 字段 |
| 成交状态变更 | 第601-607行 | 成交时初始化实付金额和欠款金额 |
| 提交数据 | 第609-637行 | 调整字段映射 |

## 注意事项
1. 实付金额和欠款金额字段在未成交时应该禁用编辑
2. 成交时，实付金额默认等于成交金额，欠款金额默认为0
3. 用户可以手动修改实付金额和欠款金额
