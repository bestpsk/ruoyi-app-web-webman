# 出库单打印预览优化计划

## 需求分析

1. **隐藏出库企业和负责员工** - 打印预览中不显示这些字段
2. **显示单位类型和换算关系** - 在打印表格中添加单位类型列，让用户能分清是主单位还是副单位

## 当前打印预览结构

```
┌────────────────────────────────────────┐
│  出库单号    出库日期                   │
│  出库类型    出库企业  ← 要隐藏          │
│  领用员工    目标企业  ← 要隐藏          │
├────────────────────────────────────────┤
│ 序号 │ 货品 │ 规格 │ 单位 │ 数量 │ ...  │
│                    ↑ 缺少单位类型标识    │
└────────────────────────────────────────┘
```

## 目标打印预览结构

```
┌────────────────────────────────────────┐
│  出库单号    出库日期                   │
│  出库类型    备注                       │
├────────────────────────────────────────┤
│ 序号 │ 货品 │ 规格 │ 单位类型 │ 数量 │ 单价 │ 金额 │
│                          ↑ 新增列       │
└────────────────────────────────────────┘
```

## 实施步骤

### 步骤1：隐藏出库企业和负责员工

修改打印预览中的 `print-info` 部分，删除以下内容：
- 第203行：出库企业显示
- 第205-208行：领用员工和目标企业显示

### 步骤2：添加单位类型列

修改打印表格，在"规格"列后添加"单位类型"列：
- 显示"主单位(盒)"或"副单位(支)"
- 如果有换算关系，可以在单位类型中显示，如"主单位(1盒=10支)"

### 步骤3：调整表格列宽

添加新列后，需要调整各列宽度以适应页面

### 打印预览信息区修改

```html
<div class="print-info">
  <div class="info-row">
    <div class="info-item"><span>出库单号：</span>{{ form.stockOutNo }}</div>
    <div class="info-item"><span>出库日期：</span>{{ form.stockOutDate }}</div>
  </div>

    <div class="info-item full"><span>备注：</span>{{ form.remark }}</div>
  </div>

```

### 打印表格修改

```html
<table class="print-table">
  <thead>
    <tr>
      <th width="40">序号</th>
      <th>货品名称</th>
      <th width="50">规格</th>
      <th width="80">单位类型</th>
      <th width="60">数量</th>
      <th width="80">单价</th>
      <th width="80">金额</th>
    </tr>
  </thead>
  <tbody>
    <tr v-for="(item, index) in form.items" :key="index">
      <td align="center">{{ index + 1 }}</td>
      <td>{{ item.productName }}</td>
      <td align="center">{{ getSpecLabel(item.spec) }}</td>
      <td align="center">
        {{ item.unitType === '1' ? getUnitLabel(item.unit) : getSpecLabel(item.spec) }}
        <span v-if="item.packQty > 1 && item.unitType === '1'" style="font-size: 10px; color: #666;">
          (1{{ getUnitLabel(item.unit) }}={{ item.packQty }}{{ getSpecLabel(item.spec) }})
        </span>
      </td>
      <td align="center">{{ item.originalQuantity || item.quantity }}</td>
      <td align="right">{{ item.salePrice }}</td>
      <td align="right">{{ item.amount }}</td>
    </tr>
  </tbody>
</table>
```
