# 首页优化设计计划

## 一、需求概述

基于用户提供的参考图片（图1：蓝色清新风格参考，图2：当前页面截图），优化首页设计，解决当前页面"不够精致、有点单调"的问题。

### 当前页面问题
1. 蓝色渐变背景色偏灰，不够通透清新
2. 菜单图标区域布局拥挤，缺少呼吸感
3. 整体缺少层次感和精致细节
4. 卡片阴影和圆角不够柔和
5. 缺少通知消息滚动区域
6. 订单列表信息展示不够清晰

### 优化方向
- 更通透的蓝色渐变（参考图1的明亮蓝色）
- 增加通知消息滚动区域
- 菜单区优化布局，增加"更多"入口
- 数据统计区增加今日/本月切换
- 订单列表：电话改为企业（门店），积分改为操作/总项次
- 整体增加精致感和高级感

## 三、组件详细设计

### 1. 顶部导航栏（优化后的蓝色渐变背景）
```
┌───────────────────────────────────────┐
│  [头像]  admin             🔔 3  ⚙   │
│          美容顾问                       │
│  晚上好，开启美好的一天！                │
└───────────────────────────────────────┘
```
- 渐变背景：linear-gradient(180deg, #5B8FF9 0%, #3D6DF7 100%)
- 高度：约 320rpx
- 底部圆角：32rpx
- 头像：圆形 96rpx，白色边框 3rpx
- 问候语：白色文字，根据时间显示

### 2. 常用菜单区（白色卡片，优化布局）
```
┌───────────────────────────────────────┐
│  🕐    📋    📅    📦                  │
│  打卡   开单    行程   订单             │
│                           [更多 >]     │
└───────────────────────────────────────┘
```
- 4列等宽布局
- 图标背景：浅蓝色圆形 #E8F0FE，直径 96rpx
- 图标颜色：主色 #3D6DF7
- 图标尺寸：40rpx
- 文字颜色：#1D2129，大小 26rpx
- 更多按钮：右侧文字链接

### 3. 通知消息滚动区域（新增）
```
┌───────────────────────────────────────┐
│  📢 [通知] 您有一条新的订单待处理...    │
│  📢 [公告] 系统将于今晚进行维护...      │
└───────────────────────────────────────┘
```
- 白色卡片背景
- 左侧喇叭图标
- 消息类型标签（通知/公告）
- 消息内容单行显示，溢出省略
- 自动滚动或手动滑动

### 4. 数据统计区（白色卡片，优化样式）
```
┌───────────────────────────────────────┐
│  今日    本月                    🔄   │
│  ─────────────────────────────────   │
│  咨询客数  成交客数  成交金额  成交项次 │
│  128↗     36↗     ¥12.8k↗   89↘    │
└───────────────────────────────────────┘
```
- 标题栏：今日/本月切换标签 + 刷新按钮
- 标签指示器：底部蓝色短线，带动画切换
- 4列数据展示
- 数字：44rpx，粗体，主色 #3D6DF7
- 趋势指示：绿色上升箭头 / 红色下降箭头（小圆点内图标）
- 分割线：浅灰色

### 5. 订单明细列表（白色卡片，优化信息展示）
```
┌───────────────────────────────────────┐
│  最近订单                      查看全部 >│
│  ─────────────────────────────────   │
│  [头像] 王雪  [已完成]       消费金额   │
│         美丽佳人美容院          88     │
│                           3项/5次 >    │
│  ─────────────────────────────────   │
│  [头像] 向晴天  [进行中]     消费金额   │
│         悦容美容会所            88     │
│                           2项/3次 >    │
└───────────────────────────────────────┘
```
- 每项高度约 140rpx
- 头像：圆形 80rpx
- 状态标签：圆角 8rpx，完成=绿色，进行中=蓝色
- 企业名称代替手机号
- 右侧金额 + 操作项次（如"3项/5次"）
- 分割线：浅灰色 1px

## 五、实施步骤

### 步骤1: 优化颜色变量
更新 `App/static/scss/global.scss` 中的颜色变量，使用更明亮的蓝色系。

### 步骤2: 重写首页组件
重写 `App/pages/index.vue`，包含以下改进：
1. 更明亮的蓝色渐变背景
2. 优化菜单区布局
3. 新增通知消息滚动区域
4. 优化数据统计区（今日/本月切换）
5. 优化订单列表（企业名称代替电话，操作项次代替积分）

### 步骤3: 测试验证
启动项目，验证首页效果是否符合预期。

## 六、注意事项

1. **uview-plus 图标**：确认使用的图标名称在 uview-plus 中可用
2. **渐变背景**：使用 CSS 渐变，确保在不同设备上显示一致
3. **通知滚动**：使用 CSS 动画或 uni-app 的 scroll-view 实现
4. **响应式**：确保在不同屏幕尺寸上显示正常
