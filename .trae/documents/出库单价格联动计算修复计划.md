# 出库单价格联动计算修复计划

## 问题分析

当前出库单的 `onUnitTypeChange` 函数逻辑不完整：

```javascript
// 当前代码（有问题）
function onUnitTypeChange(index) {
  const item = form.value.items[index]
  if (item.unitType === '1') {
    item.salePrice = item.salePrice || 0
  } else {
    item.salePrice = item.salePriceSpec || item.salePrice || 0
  }
  calcAmount(index)
}
```

**问题**：
1. 没有正确保存主单位价格 `_mainPrice`
2. 切换单位类型时没有按换算比例计算价格
3. 数量变化时虽然调用了 `calcAmount`，但价格可能不正确

## 参考入库单的正确逻辑

入库单的 `onUnitTypeChange` 函数：

```javascript
function onUnitTypeChange(index) {
  const item = form.value.items[index]
  const packQty = item.packQty || 1

  if (item.unitType === '1') {
    // 主单位：使用主单位价格
    item.purchasePrice = item._mainPrice || item.purchasePrice || 0
  } else {
    // 副单位：主单位价格 / 换算比例
    if (!item._mainPrice && item.purchasePrice) {
      item._mainPrice = item.purchasePrice
    }
    if (item._mainPrice && packQty > 0) {
      item.purchasePrice = Math.round((item._mainPrice / packQty) * 100) / 100
    }
  }

  calcAmount(index)
}
```

## 修复方案

### 步骤1：修改 onProductSelect 函数
选择货品时，保存主单位价格 `_mainPrice`

```javascript
function onProductSelect(index) {
  const product = productOptions.value.find(p => p.productId === form.value.items[index].productId)
  if (product) {
    form.value.items[index].productName = product.productName
    form.value.items[index].spec = product.spec
    form.value.items[index].unit = product.unit
    form.value.items[index].packQty = product.packQty || 1
    form.value.items[index].salePrice = product.salePrice || 0
    form.value.items[index]._mainPrice = product.salePrice || 0  // 保存主单位价格
    form.value.items[index].salePriceSpec = product.salePriceSpec || product.salePrice || 0
    form.value.items[index].inventoryQty = product.inventoryQty
    form.value.items[index].unitType = '1'
    calcAmount(index)
  }
}
```

### 步骤2：修改 onUnitTypeChange 函数
切换单位类型时，正确换算价格

```javascript
function onUnitTypeChange(index) {
  const item = form.value.items[index]
  const packQty = item.packQty || 1

  if (item.unitType === '1') {
    // 主单位：使用主单位价格
    item.salePrice = item._mainPrice || item.salePrice || 0
  } else {
    // 副单位：主单位价格 / 换算比例
    if (!item._mainPrice && item.salePrice) {
      item._mainPrice = item.salePrice
    }
    if (item._mainPrice && packQty > 0) {
      item.salePrice = Math.round((item._mainPrice / packQty) * 100) / 100
    }
  }

  calcAmount(index)
}
```

### 步骤3：确保 calcAmount 在数量变化时被调用
检查数量输入框是否绑定了 `@change="calcAmount(scope.$index)"`

## 价格换算示例

假设：1盒 = 10支，主单位价格（盒）= 100元

| unitType | 含义 | salePrice 计算 | 结果 |
|----------|------|---------------|------|
| 1 | 主单位(盒) | `_mainPrice` | 100元/盒 |
| 2 | 副单位(支) | `_mainPrice / 10` | 10元/支 |
