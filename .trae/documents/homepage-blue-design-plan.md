# 首页设计方案 - 蓝色清新简约高级风格

## 📋 项目概述

基于参考图片设计一个蓝色清新风格的首页，采用简约高级的视觉设计，使用 uview-plus 组件库和图标。

## 🎨 设计规范

### 配色方案
- **主色调**: `#3c96f3` (清新蓝)
- **渐变背景**: `linear-gradient(135deg, #4a9ff5 0%, #5b8def 100%)`
- **辅助色**: `#e8f2ff` (浅蓝背景)
- **文字颜色**:
  - 主文字: `#333333`
  - 次要文字: `#666666`
  - 辅助文字: `#999999`
- **白色卡片**: `#ffffff` + 圆角 `16rpx` + 轻阴影

### 设计原则
- 简约留白，避免过度装饰
- 卡片化布局，层次分明
- 蓝色系渐变，清新专业
- 圆角统一为 `16rpx`
- 间距统一使用 `8rpx` 的倍数

## 🏗️ 页面结构设计

### 1️⃣ 顶部导航栏区域
```
┌────────────────────────────────────────┐
│ [头像] 姓名/职位          🔔 ⚙️      │
│        副标题/欢迎语                     │
└────────────────────────────────────────┘
```

**实现要点**:
- 使用自定义导航栏（已配置 `navigationStyle: custom`）
- 渐变蓝色背景
- 左侧：圆形头像(80rpx) + 用户名(加粗) + 职位标签
- 右侧：消息图标(u-icon: bell) + 设置图标(u-icon: setting)
- 底部显示欢迎语或日期

**组件使用**:
- `u-avatar` 显示头像
- `u-icon` 显示图标
- `u-badge` 消息角标

### 2️⃣ 常用菜单区域（Quick Actions）
```
┌────────────────────────────────────────┐
│   📋打卡    💰开单    📅行程    📦订单   │
└────────────────────────────────────────┘
```

**实现要点**:
- 白色卡片容器，圆角 `16rpx`
- 4个菜单项横向排列，均分宽度
- 每个菜单：图标(56rpx) + 文字(24rpx)
- 图标使用圆形浅蓝背景(`#e8f2ff`)
- 点击反馈效果（缩放动画）

**uview-plus 图标映射**:
- 打卡: `clock` 或 `edit-pen`
- 开单: `order` 或 `yen-circle`
- 行程: `calendar` 或 `map`
- 订单: `list` 或 `file-text`

### 3️⃣ 数据统计区域（Statistics Card）
```
┌────────────────────────────────────────┐
│  📊 今日数据概览                         │
│ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐   │
│ │咨询客数│ │成交客数│ │成交金额│ │成交项次│  │
│ │  128  │ │  36   │ │¥12.8k│ │  89   │  │
│ └──────┘ └──────┘ └──────┘ └──────┘   │
└────────────────────────────────────────┘
```

**实现要点**:
- 白色卡片容器
- 标题："今日数据概览" 或 "数据统计"
- 4列网格布局，每列显示：
  - 标签文字(24rpx, 灰色)
  - 数值(36rpx, 加粗, 蓝色或深色)
- 可选：环比箭头指示（上升绿色/下降红色）

**数据字段**:
- consultCount: 咨询客数
- dealCount: 成交客数
- dealAmount: 成交金额
- dealItems: 成交项次

### 4️⃣ 订单明细列表（Order List）

```
│  📋 最近订单              查看全部 >    │
│ ┌───────────────────────────────────┐ │

│ │    消费金额: ¥2560  |  积分: 240   │ │
│ └───────────────────────────────────┘ │
└────────────────────────────────────────┘
```

**实现要点**:
- 白色卡片容器
- 标题行："最近订单" + "查看全部"链接
- 列表项布局：
  - 左侧：顾客头像(小) + 姓名 + 手机号(脱敏)
  - 中间：消费金额(突出显示)
  - 右侧：操作按钮("查看"或箭头)
- 底部显示积分或其他附加信息
- 分隔线或卡片间隔

**组件使用**:
- `u-list` + `u-list-item` 或自定义 scroll-view
- `u-avatar` 顾客头像
- `u-tag` 订单状态标签
- `u-icon` 箭头图标

## 🔧 实施步骤

**文件**: `AppV3/src/api/home.js`

定义以下接口：
- `getHomeData()` - 获取首页综合数据（用户信息、统计数据）
- `getRecentOrders()` - 获取最近订单列表
- `getTodayStats()` - 获取今日统计数据

### Step 2: 创建顶部导航栏组件
**文件**: `AppV3/src/components/home/HeaderNav.vue`

功能：
- 显示用户头像、姓名、职位
- 消息图标（带未读数量角标）
- 设置图标入口
- 自适应状态栏高度
- 渐变蓝色背景

### Step 3: 创建常用菜单组件
**文件**: `AppV3/src/components/home/QuickMenu.vue`

功能：
- 4个快捷操作按钮（打卡、开单、行程、订单）
- 图标 + 文字布局
- 点击跳转或触发事件
- 触摸反馈动画

### Step 4: 创建数据统计卡片组件
**文件**: `AppV3/src/components/home/StatisticsCard.vue`

功能：
- 4列数据展示
- 数值动态渲染
- 可选的环比指标
- 数据加载状态处理

### Step 5: 创建订单列表组件
**文件**: `AppV3/src/components/home/OrderList.vue`

功能：
- 订单列表展示
- 下拉刷新 / 上拉加载更多
- 空状态展示
- 点击跳转订单详情

### Step 6: 整合首页主文件
**文件**: `AppV3/src/pages/index.vue`

整合所有组件：
- 引入并组合上述4个组件
- 页面生命周期管理
- 数据加载逻辑
- 下拉刷新实现
- 整体页面样式协调

### Step 7: 样式优化与细节打磨
- 统一间距规范
- 过渡动画优化
- 加载状态处理
- 异常状态展示
- 适配不同屏幕尺寸

## 🎯 关键技术点

### 1. 自定义导航栏适配
```javascript
// 获取系统信息，动态计算状态栏高度
const systemInfo = uni.getSystemInfoSync()
const statusBarHeight = systemInfo.statusBarHeight
```

### 2. uview-plus 图标使用
```vue
<u-icon name="clock" size="28" color="#3c96f3" />
<u-icon name="bell" size="24" color="#fff" />
```

### 3. 渐变背景实现
```scss
.header-bg {
  background: linear-gradient(135deg, #4a9ff5 0%, #5b8def 100%);
}
```

### 4. 卡片阴影效果
```scss
.card {
  background: #fff;
  border-radius: 16rpx;
  box-shadow: 0 4rpx 16rpx rgba(60, 150, 243, 0.08);
  margin: 20rpx;
  padding: 30rpx;
}
```

2. **权限控制**: 部分菜单需根据用户角色显示/隐藏
