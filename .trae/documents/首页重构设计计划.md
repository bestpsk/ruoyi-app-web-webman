# 首页重构设计计划

## 一、设计目标

基于用户提供的参考图片，重构移动端首页，打造蓝色清新、简约高级的视觉风格。

### 页面结构
```
┌───────────────────────────────────┐
│ 顶部导航栏：头像+姓名/职位 | 消息+设置图标 │
├───────────────────────────────────┤
│ 常用菜单区：打卡 | 开单 | 行程 | 订单  │更多 │
├───────────────────────────────────┤
│ 数据统计区：今日和本月，咨询客数 | 成交客数 | 成交金额 | 成交项次 │
├───────────────────────────────────┤
│ 订单明细列表：顾客信息 + 金额 + 操作/总项次 │
└───────────────────────────────────┘
```

## 三、组件详细设计

### 1. 顶部导航栏（蓝色渐变背景）
```
┌───────────────────────────────────────┐
│  [头像]  admin             🔔 3  ⚙   │
│          美容顾问                       │
│  晚上好，开启美好的一天！                │
└───────────────────────────────────────┘
```
- 渐变背景：linear-gradient(135deg, #6E93F7 0%, #4A7BF7 100%)
- 高度：约 340rpx
- 底部圆角：40rpx（内凹效果）
- 头像：圆形 100rpx，白色边框 4rpx
- 问候语：白色文字，根据时间显示早上好/下午好/晚上好

### 2. 常用菜单区（白色卡片）
```
┌───────────────────────────────────────┐
│  🕐    📋    📅    📦                  │
│  打卡   开单    行程   订单             │
│                           [更多 >]     │
└───────────────────────────────────────┘
```
- 4列等宽布局
- 图标背景：浅蓝色圆形 #EBF1FF，直径 100rpx
- 图标颜色：主色 #4A7BF7
- 图标尺寸：44rpx
- 文字颜色：#1D2129，大小 26rpx
- 更多按钮：右侧文字链接

### 3. 数据统计区（白色卡片）
```
┌───────────────────────────────────────┐
│  今日数据概览                    🔄   │
│  ─────────────────────────────────   │
│  咨询客数  成交客数  成交金额  成交项次 │
│  128↗     36↗     ¥12.8k↗   89↘    │
└───────────────────────────────────────┘
```
- 标题栏：标题 + 刷新按钮
- 4列数据展示
- 数字：48rpx，粗体，主色 #4A7BF7
- 趋势指示：绿色上升箭头 / 红色下降箭头
- 分割线：浅灰色

### 4. 订单明细列表（白色卡片）
```
┌───────────────────────────────────────┐
│  最近订单                      查看全部 >│
│  ─────────────────────────────────   │
│  [头像] 王雪  [已完成]       消费金额   │
│         187****9766            88     │
│                           积分 12 >    │
│  ─────────────────────────────────   │
│  [头像] 向晴天  [进行中]     消费金额   │
│         187****9766            88     │
│                           积分 12 >    │
└───────────────────────────────────────┘
```
- 每项高度约 140rpx
- 头像：圆形 80rpx
- 状态标签：圆角 12rpx，完成=绿色，进行中=蓝色
- 金额：右对齐，粗体 32rpx
- 分割线：浅灰色 1px

## 六、实施步骤

### 步骤1: 定义全局样式变量
在 `global.scss` 中添加颜色变量：
```scss
$primary-color: #4A7BF7;
$primary-gradient-start: #6E93F7;
$primary-gradient-end: #4A7BF7;
$bg-color: #F4F6FB;
$card-bg: #FFFFFF;
$text-primary: #1D2129;
$text-secondary: #4E5969;
$text-auxiliary: #86909C;
$success-color: #00B42A;
$info-color: #3491FA;
$danger-color: #F53F3F;
$divider-color: #E5E6EB;
$icon-bg: #EBF1FF;
```

### 步骤2: 重写首页组件
重写 `App/pages/index.vue`，包含以下模块：
1. 顶部蓝色渐变导航栏
2. 常用菜单区（打卡/开单/行程/订单）
3. 数据统计区（今日/本月切换 + 四项数据）
4. 最近订单列表

### 步骤3: 模拟数据
先使用模拟数据展示效果，后续对接后端接口。
