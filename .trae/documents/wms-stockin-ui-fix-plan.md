# 入库管理优化计划（简化版）

## 需求变更

### 有效期管理简化
**原方案**：货品设置保质期天数 → 入库填生产日期 → 自动计算有效期至
**新方案**：入库时**直接填写**生产日期和有效期至（更灵活）

**理由**：
- 很多产品只有生产日期和有效期至，没有固定保质期
- 不同批次可能有不同有效期
- 直接填写更简单实用
