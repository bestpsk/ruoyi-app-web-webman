### 单次价当前显示逻辑

**代码位置**：[第65行](d:/fuchenpro/AppV3/src/pages/business/sales/order.vue#L65)

```html
<text class="form-value auto-hint">
  <u-icon name="info-circle" size="12" color="#C9CDD4"></u-icon>
  ¥{{ calcUnitPrice(item) }}
  <text class="auto-tag">自动</text>
</text>
```

**显示效果**：
```
ℹ️ ¥0.00 自动
```

**含义说明**：
- `info-circle` 图标：提示这是一个计算字段
- `¥0.00`：单次价（成交金额 ÷ 次数）
- `自动` 标签：表示该值是系统自动计算的，非手动输入

## 问题诊断与解决方案

### 问题1：图标是否需要统一？

**现状评估**：
- 所有图标都是 uView Plus 支持的标准图标 ✅
- 图标风格一致（都使用 u-icon 组件）
- 颜色使用合理（主题色、灰色、红色等）

**结论**：当前图标已经符合要求，无需大规模修改

### 问题2：单次价显示是否需要优化？

**可能的问题**：
1. 信息过于复杂（图标 + 金额 + "自动"文字）
2. 视觉上与其他输入框不协调
3. "自动"标签的含义不够直观

### 优化方案

#### 方案A：简化单次价显示（推荐）✅

保留核心信息，移除冗余元素：

```html
<!-- 方案A1：只保留金额 -->
<text class="form-value">¥{{ calcUnitPrice(item) }}</text>

<!-- 方案A2：保留金额 + 淡化样式 -->
<text class="form-value auto-hint">¥{{ calcUnitPrice(item) }} <text style="font-size: 20rpx; color: #C9CDD4;">自动</text></text>

<!-- 方案A3：保留图标 + 金额，移除"自动"文字 -->
<text class="form-value auto-hint"><u-icon name="info-circle" size="12" color="#C9CDD4"></u-icon> ¥{{ calcUnitPrice(item) }}</text>
```

#### 方案B：保持现状但优化样式

如果用户觉得当前显示可以接受，只需微调：

```css
.auto-tag {
  font-size: 20rpx;  /* 缩小字体 */
  color: #C9CDD4;    /* 使用更浅的颜色 */
  margin-left: 4rpx;
}
```

## 注意事项

1. **保持一致性**：单次价是只读字段，样式应与欠款金额保持一致
2. **用户体验**："自动"标签的目的是告知用户这是计算值，如果移除需确保用户不会困惑
3. **功能不变**：无论UI如何调整，单次价的计算逻辑保持不变
