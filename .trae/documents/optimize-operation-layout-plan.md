# 操作Tab布局优化 - 持卡明细与操作表单并排显示

## 问题
当前操作Tab的布局是纵向排列：
1. 持卡明细（多个套餐组，每个组有header+表格，占大量纵向空间）
2. 持卡操作表单（选了品项后才出现，被推到最下方）
3. 体验操作表单

套餐多时，操作表单被推到页面最底部，需要大量滚动才能操作。

## 方案：左右分栏布局

将操作Tab改为**左右两栏**：
- **左栏（60%宽度）**：持卡明细，可滚动
- **右栏（40%宽度）**：操作表单，sticky定位，始终可见

```
┌──────────────────────────────┬─────────────────────┐
│  持卡明细                     │  持卡操作            │
│  ┌────────────────────────┐  │  ┌─────────────────┐ │
│  │ 套餐A  已成交  ¥3000   │  │  │ 品项 | 次数|金额 │ │
│  │ ☑美白  ☑补水  ☑祛斑   │  │  │ 美白  1   ¥200  │ │
│  └────────────────────────┘  │  │ 补水  1   ¥150  │ │
│  ┌────────────────────────┐  │  ├─────────────────┤ │
│  │ 套餐B  已成交  ¥5000   │  │  │ 操作时间: ...   │ │
│  │ ☑紧致  ☑抗衰          │  │  │ 操作人: ...     │ │
│  └────────────────────────┘  │  │ 满意度: ★★★★★  │ │
│  ┌────────────────────────┐  │  │ 顾客反馈: ...   │ │
│  │ 套餐C  已用完  ¥2000   │  │  │ [提交持卡操作]  │ │
│  │ 紧致  抗衰             │  │  └─────────────────┘ │
│  └────────────────────────┘  │                      │
│  [显示用完] [+体验操作]      │  或 体验操作表单      │
└──────────────────────────────┴─────────────────────┘
```

## 具体修改

### 文件：`d:\fuchenpro\front\src\views\business\sales\index.vue`

#### 1. 操作Tab模板改为左右分栏
- 外层 `el-row` + 两个 `el-col`
- 左栏 `el-col :span="14"`：持卡明细
- 右栏 `el-col :span="10"`：操作表单（持卡操作 + 体验操作），使用 `position: sticky; top: 0`

#### 2. 持卡明细精简显示
- 每个套餐组改为**可折叠**（el-collapse），默认展开
- 套餐header更紧凑：名称+状态+金额一行显示
- 品项表格精简列宽

#### 3. 操作表单sticky定位
- 右栏使用 `position: sticky; top: 16px; align-self: flex-start`
- 无论左栏多长，右栏始终可见

#### 4. 无选中品项时右栏显示提示
- "请从左侧选择品项进行操作"

## 优势
1. 操作表单始终可见，无需滚动
2. 持卡明细和操作表单可同时查看
3. 套餐可折叠，减少纵向占用
4. 视觉上更平衡，信息密度更高
