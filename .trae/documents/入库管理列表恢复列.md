# 入库管理列表 — 恢复货品名称和供货商列

## 问题

入库管理列表表格缺少 **货品名称** 和 **供货商** 列。

## 当前列结构
```
入库单号 | 入库类型 | 总数量 | 总金额 | 入库日期 | 操作人 | 状态 | 创建时间 | 有效期至 | 操作
```

## 目标列结构
```
入库单号 | 货品名称 | 供货商 | 入库类型 | 总数量 | 总金额 | 入库日期 | 状态 | 创建时间 | 有效期至 | 操作
```

## 实施方案

### 在入库单号后面添加两列

**货品名称列**：
- 从 `getList` 中已获取的详情数据中提取第一个货品名称
- 如果有多个货品，显示第一个 + "等N个"

**供货商列**：
- 同样从详情数据中提取第一个供货商名称

### 数据来源

`getList` 方法已经通过 `getStockIn(row.stockInId)` 获取了每条记录的详情（包括 items），在补全 display 字段时可以同时保存货品名称和供货商。
