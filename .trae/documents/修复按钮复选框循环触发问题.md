# 修复按钮复选框循环触发问题

## 问题分析

当前实现有两个 watch 互相触发，形成循环：

1. 用户点击按钮复选框 → `buttonCheckState` 变化
2. 第二个 watch 触发 → 调用 `menuRef.value.setChecked`
3. menuRef 选中状态变化 → 第一个 watch 触发
4. 第一个 watch 更新 `buttonCheckState` → 又触发第二个 watch
5. 循环...

## 解决方案

使用标志位防止循环触发。

### 实施步骤

**文件**: `front/src/views/system/role/index.vue`

#### 步骤1：添加标志位

```javascript
const isUpdatingFromTree = ref(false)
```

#### 步骤2：修改两个 watch

```javascript
watch(() => menuRef.value?.getCheckedKeys(), (keys) => {
  if (keys && !isUpdatingFromTree.value) {
    const newState = {}
    keys.forEach(id => {
      newState[id] = true
    })
    buttonCheckState.value = newState
  }
}, { deep: true })

watch(buttonCheckState, (newState, oldState) => {
  if (!menuRef.value) return
  isUpdatingFromTree.value = true
  Object.keys(newState).forEach(id => {
    const wasChecked = oldState?.[id]
    const isNowChecked = newState[id]
    if (wasChecked !== isNowChecked) {
      menuRef.value.setChecked(Number(id), isNowChecked, false)
    }
  })
  nextTick(() => {
    isUpdatingFromTree.value = false
  })
}, { deep: true })
```

## 预期效果

点击按钮复选框时，不会触发循环，菜单选中状态保持稳定。
