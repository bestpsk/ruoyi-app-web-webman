# 员工配置休息日功能修复计划

## 一、问题分析

### 问题1：保存休息日期后数据丢失
**现象**：选好休息日后点击保存，再次打开配置弹窗，选中状态消失

**根因**：前端API发送数据格式与后端接收不匹配

| 位置 | 当前代码 | 问题 |
|------|---------|------|
| 前端 API | `data: restDates` (数组) | 发送的是裸数组 `['2026-04-30']` |
| 后端 Controller | `$request->post('restDates', [])` | 期望表单字段 `restDates=xxx` |

**数据流断裂**：POST body是JSON数组，没有key名称，后端无法通过`post('restDates')`获取

### 问题2：员工配置表格缺少休息日列
**需求**：在"部门"列后面新增一列"休息日"，用tag显示已配置的休息日期

---

## 二、修复方案

### 修复1：前端API修正数据格式 ⏱️ 3分钟

**文件**：`d:\fuchenpro\front\src\api\business\employeeConfig.js`

```javascript
// 修改前（错误）
export function saveRestDates(userId, restDates) {
  return request({
    url: '/business/employeeConfig/saveRestDates',
    method: 'post',
    params: { userId },
    data: restDates  // ❌ 裸数组，没有key
  })
}

// 修改后（正确）
export function saveRestDates(userId, restDates) {
  return request({
    url: '/business/employeeConfig/saveRestDates',
    method: 'post',
    data: { userId, restDates }  // ✅ 包装为对象
  })
}
```

### 修复2：后端Controller适配新格式 ⏱️ 3分钟

**文件**：`d:\fuchenpro\webman\app\controller\business\BizEmployeeConfigController.php`

```php
public function saveRestDates(Request $request)
{
    $data = convert_to_snake_case($request->post());
    $userId = $data['user_id'];
    $restDates = $data['rest_dates'] ?? [];
    // ... 后续逻辑不变
}
```

### 修复3：员工配置列表增加休息日数据 ⏱️ 5分钟

**文件**：`d:\fuchenpro\webman\app\service\BizEmployeeConfigService.php`

修改`selectConfigList`方法，返回时包含`rest_dates`字段：

```php
foreach ($result->items() as $item) {
    // 现有职位补充逻辑...
    
    // 补充休息日期
    if ($item->rest_dates) {
        $item->rest_dates = json_decode($item->rest_dates, true) ?: [];
    } else {
        $item->rest_dates = [];
    }
}
```

### 修复4：前端表格新增休息日列 ⏱️ 5分钟

**文件**：`d:\fuchenpro\front\src\views\business\schedule\index.vue`

在部门列后面、是否可排班列前面添加：

```vue
<el-table-column label="休息日" min-width="180">
  <template #default="scope">
    <el-tag 
      v-for="(date, idx) in (scope.row.restDates || [])" 
      :key="idx" 
      size="small" 
      type="danger"
      class="mx-1"
    >
      {{ date.slice(5) }}
    </el-tag>
    <span v-if="!scope.row.restDates?.length" class="text-gray-400">未配置</span>
  </template>
</el-table-column>
```
