# 进销存管理前端页面实施计划

## 问题原因

后端菜单配置中 `component` 字段指向 `wms/supplier/index`、`wms/product/index` 等路径，但前端 `front/src/views/wms/` 目录下没有对应的 Vue 页面组件，导致点击菜单显示空白页。

前端动态路由机制通过 `import.meta.glob('./../../views/**/*.vue')` 扫描所有视图文件，只有文件存在时才能正确加载。

---

## 需要创建的文件

### 一、API 文件（`front/src/api/wms/`）

| 文件 | 说明 |
|------|------|
| `supplier.js` | 供货商 CRUD + 搜索 |
| `product.js` | 货品 CRUD + 搜索 |
| `stockIn.js` | 入库单 CRUD + 确认 |
| `stockOut.js` | 出库单 CRUD + 确认 |
| `inventory.js` | 库存列表 + 预警 + 详情 |
| `stockCheck.js` | 盘点单 CRUD + 确认 + 加载库存 |
| `report.js` | 入库汇总 + 出库汇总 + 库存周转 + 货品流水 |

### 二、Vue 页面组件（`front/src/views/wms/`）

| 文件 | 说明 | 页面类型 |
|------|------|----------|
| `supplier/index.vue` | 供货商管理 | 标准 CRUD |
| `product/index.vue` | 货品管理 | 标准 CRUD |
| `stockIn/index.vue` | 入库管理 | 主从表 CRUD + 确认 |
| `stockOut/index.vue` | 出库管理 | 主从表 CRUD + 确认 |
| `inventory/index.vue` | 库存查看 | 只读列表 + 预警筛选 |
| `stockCheck/index.vue` | 库存盘点 | 主从表 CRUD + 确认 |
| `report/index.vue` | 进销存报表 | 统计报表（4个Tab） |

## 实施步骤

### 第1步：创建7个API文件
- 遵循项目现有 API 定义模式（参考 enterprise.js）
- 每个文件导出 CRUD 函数 + 业务特定函数

### 第2步：创建供货商管理页面 `wms/supplier/index.vue`
- 标准 CRUD 页面：搜索表单 + 操作按钮 + 表格 + 分页 + 弹窗表单
- 搜索字段：供货商名称、联系人、状态
- 表格列：供货商名称、联系人、联系电话、地址、合作起始日期、状态、创建时间、操作
- 表单字段：供货商名称、联系人、联系电话、地址、合作起始日期、状态、备注

### 第3步：创建货品管理页面 `wms/product/index.vue`
- 标准 CRUD 页面
- 搜索字段：品名、货品编码、类别（字典 biz_product_category）、供货商、状态
- 表格列：货品编码、品名、供货商、规格、类别（字典标签）、单位、进货价、出货价、预警数量、状态、操作
- 表单字段：品名、货品编码、供货商（下拉搜索）、规格、类别（字典下拉）、单位、进货价、出货价、预警数量、状态、备注

### 第4步：创建入库管理页面 `wms/stockIn/index.vue`
- 主从表 CRUD + 确认功能
- 搜索字段：入库单号、入库类型（字典 biz_stock_in_type）、供货商、状态（字典 biz_doc_status）、入库日期范围
- 表格列：入库单号、入库类型（字典标签）、供货商、总数量、总金额、入库日期、操作人、状态（字典标签）、操作（查看/修改/删除/确认）
- 新增/修改弹窗：主表字段 + 可编辑明细表格（选择货品、输入数量、单价，自动计算金额）
- 确认操作：二次确认后调用确认接口
- 已确认的单据不可修改/删除

### 第5步：创建出库管理页面 `wms/stockOut/index.vue`
- 主从表 CRUD + 确认功能
- 搜索字段：出库单号、出库类型（字典 biz_stock_out_type）、出库企业、状态、出库日期范围
- 表格列：出库单号、出库类型、出库企业、负责员工、总数量、总金额、出库日期、状态、操作
- 新增/修改弹窗：主表字段（出库企业下拉、负责员工）+ 可编辑明细表格
- 确认操作同入库

### 第6步：创建库存查看页面 `wms/inventory/index.vue`
- 只读列表 + 预警筛选
- 搜索字段：品名、货品编码、类别、是否预警
- 表格列：货品编码、品名、规格、类别、单位、当前库存、预警数量、库存状态（正常/预警）、最后入库时间、最后出库时间
- 无新增/修改/删除操作

### 第7步：创建库存盘点页面 `wms/stockCheck/index.vue`
- 主从表 CRUD + 确认功能
- 搜索字段：盘点单号、状态、盘点日期范围
- 表格列：盘点单号、盘点日期、总数量、差异数量、操作人、状态、操作
- 新增弹窗：先加载库存数据，填写实际盘点数量，自动计算差异

### 第8步：创建进销存报表页面 `wms/report/index.vue`
- 4个Tab页：入库汇总、出库汇总、库存周转、货品收发存明细
- 每个Tab有独立的搜索条件和数据表格
- 入库汇总：按时间范围、供货商、类别统计
- 出库汇总：按时间范围、企业、类别统计
- 库存周转：期初+入库-出库=期末
- 货品流水：选择货品+时间范围查看出入库明细

## 关键设计说明

1. **字典使用**：通过 `useDict()` 获取字典数据，搜索表单用 `el-select`，表格用 `dict-tag` 组件
2. **权限控制**：按钮使用 `v-hasPermi` 指令，权限标识与后端菜单配置一致
3. **主从表页面**：入库/出库/盘点使用可编辑明细表格，支持动态添加/删除行
4. **确认机制**：入库/出库/盘点单据确认后不可修改/删除，确认前二次弹窗提示
5. **供货商/货品下拉搜索**：使用 `el-select` + `remote-method` 实现远程搜索
6. **出库关联企业和员工**：出库企业使用已有的企业搜索接口，员工从系统用户中选择
