# 入库单和出库单打印模板优化计划

## 当前问题分析

### 入库单打印
- **数量列**：只显示纯数字（如 `10`），没有单位标签
- **单位列**：只显示单位名称（如"箱"或"支"），缺少换算关系
- **合计行**：总数量只显示纯数字，没有单位
- **缺少供货商列**：items 数据中有 supplierName 但打印没显示

### 出库单打印
- **数量列**：只显示纯数字，没有单位标签
- **单位列**：只显示单位名称，缺少换算关系
- **合计行**：总数量只显示纯数字，没有单位

### 共同问题
- 打印表格的列与实际数据不匹配，缺少关键信息
- 数量没有带单位，不够直观

## 优化方案

### 入库单打印表格

**新列结构**：

| 序号 | 货品名称 | 供货商 | 规格 | 换算 | 数量 | 进货单价 | 金额 |
|------|----------|--------|------|------|------|----------|------|
| 1 | GCS-p7 | XX供货商 | 支 | 1箱=10支 | 1箱(10支) | 25.80 | 258.00 |

- **供货商**：新增列，显示 item.supplierName
- **换算**：新增列，显示换算关系如 `1箱=10支`
- **数量**：使用 formatItemQuantity 方法，显示为 `1箱（10支）` 格式
- **合计行**：总数量也使用 formatQuantity 方法显示

### 出库单打印表格

**新列结构**：

| 序号 | 货品名称 | 规格 | 换算 | 数量 | 出货单价 | 金额 |
|------|----------|------|------|------|----------|------|
| 1 | GCS-p7 | 支 | 1箱=10支 | 1箱(10支) | 25.80 | 258.00 |

- **换算**：新增列，显示换算关系
- **数量**：使用 formatItemQuantity 方法，显示为 `1箱（10支）` 格式
- **合计行**：总数量也使用 formatQuantity 方法显示

### formatItemQuantity 方法

用于打印明细行的数量格式化（与列表的 formatQuantity 逻辑一致，但参数来源不同）：

```javascript
function formatItemQuantity(item) {
  const packQty = item.packQty || 1
  const unitLabel = getUnitLabel(item.unit)
  const specLabel = getSpecLabel(item.spec)
  const qty = item.quantity || 0
  if (packQty > 1 && specLabel) {
    const mainQty = qty / packQty
    const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
    return mainQtyStr + unitLabel + '（' + qty + specLabel + '）'
  } else if (unitLabel) {
    return qty + unitLabel
  } else {
    return qty
  }
}
```

### formatTotalQuantity 方法

用于合计行的总数量格式化（需要从 items 中获取第一条的换算信息）：

```javascript
function formatTotalQuantity(form) {
  const firstItem = form.items?.[0]
  if (!firstItem) return form.totalQuantity || 0
  const packQty = firstItem.packQty || 1
  const unitLabel = getUnitLabel(firstItem.unit)
  const specLabel = getSpecLabel(firstItem.spec)
  const totalQty = form.totalQuantity || 0
  if (packQty > 1 && specLabel) {
    const mainQty = totalQty / packQty
    const mainQtyStr = Number.isInteger(mainQty) ? mainQty : mainQty.toFixed(1).replace(/\.0$/, '')
    return mainQtyStr + unitLabel + '（' + totalQty + specLabel + '）'
  } else if (unitLabel) {
    return totalQty + unitLabel
  } else {
    return totalQty
  }
}
```

## 实施步骤

### 步骤1：修改出库单打印模板 - stockOut/index.vue
1. 打印表格 thead 改为新列结构：序号 | 货品名称 | 规格 | 换算 | 数量 | 出货单价 | 金额
2. 打印表格 tbody 每行数据：
   - 规格列：`getSpecLabel(item.spec)`
   - 换算列：`item.packQty > 1 ? '1' + getUnitLabel(item.unit) + '=' + item.packQty + getSpecLabel(item.spec) : '-'`
