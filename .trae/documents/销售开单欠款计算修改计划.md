# 销售开单-欠款计算和开单记录显示修改计划

## 需求概述
1. **欠款金额计算逻辑修改**：欠款应该默认等于成交金额 - 实付金额
2. **开单记录表格添加列**：在开单记录中显示实付金额和欠款金额

## 涉及文件
- `d:\fuchenpro\front\src\views\business\sales\index.vue`

## 详细修改步骤

### 步骤1：修改欠款金额计算逻辑

**问题**：当前欠款金额固定为0，应该动态计算为：成交金额 - 实付金额

**方案**：将欠款金额列改为只读显示，自动计算 `dealAmount - paidAmount`

**修改位置**：第120-124行

**当前代码：**
```vue
<el-table-column label="欠款金额" width="140">
  <template #default="scope">
    <el-input-number v-model="scope.row.debtAmount" :min="0" :precision="2" :disabled="!scope.row.isDeal" controls-position="right" style="width: 100%" />
  </template>
</el-table-column>
```

**修改为：**
```vue
<el-table-column label="欠款金额" width="100" align="center">
  <template #default="scope">
    <span>{{ scope.row.isDeal ? (scope.row.dealAmount - scope.row.paidAmount).toFixed(2) : '0.00' }}</span>
  </template>
</el-table-column>
```

### 步骤2：修改成交状态变更处理函数

**修改位置**：第606-616行

**当前代码：**
```javascript
function handleDealChange(index, val) {
  if (val === 1) {
    orderItems.value[index].dealAmount = orderItems.value[index].planPrice
    orderItems.value[index].paidAmount = orderItems.value[index].planPrice
    orderItems.value[index].debtAmount = 0
  } else {
    orderItems.value[index].dealAmount = 0
    orderItems.value[index].paidAmount = 0
    orderItems.value[index].debtAmount = 0
  }
}
```

**修改为：**
```javascript
function handleDealChange(index, val) {
  if (val === 1) {
    orderItems.value[index].dealAmount = orderItems.value[index].planPrice
    orderItems.value[index].paidAmount = orderItems.value[index].planPrice
  } else {
    orderItems.value[index].dealAmount = 0
    orderItems.value[index].paidAmount = 0
  }
}
```

### 步骤3：修改数据初始化

**修改位置**：第600-604行

**当前代码：**
```javascript
function addOrderItemRow() {
  orderItems.value.push({
    productName: '', quantity: 1, planPrice: 0, isDeal: 0, dealAmount: 0, paidAmount: 0, debtAmount: 0
  })
}
```

**修改为：**
```javascript
function addOrderItemRow() {
  orderItems.value.push({
    productName: '', quantity: 1, planPrice: 0, isDeal: 0, dealAmount: 0, paidAmount: 0
  })
}
```

### 步骤4：在开单记录表格中添加实付金额和欠款金额列

**修改位置**：第336-338行（成交金额列后面）

**当前代码：**
```vue
<el-table-column label="成交金额" min-width="110" align="right">
  <template #default="scope">{{ Number(scope.row.dealAmount || 0).toFixed(2) }}</template>
</el-table-column>
<el-table-column label="订单状态" min-width="90" align="center">
```

**修改为：**
```vue
<el-table-column label="成交金额" min-width="110" align="right">
  <template #default="scope">{{ Number(scope.row.dealAmount || 0).toFixed(2) }}</template>
</el-table-column>
<el-table-column label="实付金额" min-width="110" align="right">
  <template #default="scope">{{ Number(scope.row.paidAmount || 0).toFixed(2) }}</template>
</el-table-column>
<el-table-column label="欠款金额" min-width="110" align="right">
  <template #default="scope">{{ Number(scope.row.debtAmount || 0).toFixed(2) }}</template>
</el-table-column>
<el-table-column label="订单状态" min-width="90" align="center">
```

## 修改总结

| 修改项 | 位置 | 说明 |
|--------|------|------|
| 欠款金额列 | 第120-124行 | 改为自动计算显示，不再手动输入 |
| 成交状态变更 | 第606-616行 | 移除 debtAmount 的手动设置 |
| 数据初始化 | 第600-604行 | 移除 debtAmount 字段初始化 |
| 开单记录表格 | 第336行后 | 添加实付金额和欠款金额列 |

## 注意事项
1. 欠款金额 = 成交金额 - 实付金额，自动计算，用户不可编辑
2. 开单记录中的欠款金额从后端返回的 `debtAmount` 字段读取
3. 前端不再单独维护 debtAmount，由后端计算存储
