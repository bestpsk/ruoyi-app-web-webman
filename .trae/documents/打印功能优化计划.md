# 打印功能优化计划

## 问题分析

1. **跳转新窗口问题**：当前使用 `window.open('', '_blank')` 打开新窗口，会被浏览器拦截或跳转
2. **预览页面空白**：可能是浏览器安全限制或 `window.print()` 执行时机问题

## 解决方案

改为在当前页面使用弹窗方式预览打印内容，在弹窗内显示打印预览，并提供"确认打印"按钮。

### 方案设计

```

│  打印预览                                          [关闭]   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐   │
│   │                                                     │   │
│   │              入 库 单 / 出 库 单                     │   │
│   │              （打印内容预览）                        │   │
│   │                                                     │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                    [确认打印]  [取消]                        │
└─────────────────────────────────────────────────────────────┘
```

## 技术实现

### 打印预览弹窗结构

```html
<el-dialog title="打印预览" v-model="printPreview" width="900px" append-to-body>
  <div class="print-preview-content">
    <!-- 打印内容 -->
  </div>
  <template #footer>
    <el-button type="primary" @click="confirmPrint">确认打印</el-button>
    <el-button @click="printPreview = false">取消</el-button>
  </template>
</el-dialog>
```

### 打印执行方式

使用 CSS `@media print` 隐藏非打印内容，调用 `window.print()` 打印当前页面：

```javascript
function confirmPrint() {
  const printContent = document.getElementById('print-content')
  const printWindow = window.open('', '_blank')
  printWindow.document.write(`
    <html>
    <head>
      <title>打印</title>
      <style>/* 打印样式 */</style>
    </head>
    <body>${printContent.innerHTML}</body>
    </html>
  `)
  printWindow.document.close()
  printWindow.focus()
  printWindow.print()
  printWindow.close()
}
```

或者使用 iframe 方式打印：

```javascript
function confirmPrint() {
  const iframe = document.createElement('iframe')
  iframe.style.display = 'none'
  document.body.appendChild(iframe)
  iframe.contentDocument.write(printHtml)
  iframe.contentDocument.close()
  iframe.contentWindow.print()
  document.body.removeChild(iframe)
}
```
