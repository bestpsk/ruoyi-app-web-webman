# 行程安排功能全面修复与优化计划

## 一、问题描述汇总

根据用户反馈和截图分析，当前存在以下主要问题：

### 1.1 日历视图问题
- ❌ 连续行程**没有真正合并单元格**，显示的是小色块而非占满整个合并区域
- ❌ 拖动选择**颜色提示不生效**
- ❌ 鼠标悬停**提示（tooltip）消失了**

### 1.2 添加/修改行程表单问题
- ❌ 企业显示为数字"1"而不是企业名称
- ❌ 状态显示为"0"而不是状态文字
- ❌ 布局不合理，需要优化：员工和企业各占一行，下店目的和状态用选项卡

### 1.3 排班列表问题
- ❌ 单个员工的行程**分成多行显示**，应该合并成一行
- ❌ 缺少**日期列**显示具体哪几天
- ❌ 状态显示都是"0"
- ❌ 列宽不均匀（有的特别宽，有的特别窄）

### 1.4 拼音转换问题
- ❌ 当前使用汉字映射表，**汉字数量不够容易漏字**
- ❌ 需要**使用专业的拼音插件**

### 1.5 员工配置Tab页问题
- ❌ 员工数据**不显示**（暂无数据）

## 二、详细修复方案

### 2.1 日历视图单元格合并修复

**当前问题原因：**
el-table 使用 `:rowspan` 和 `:colspan` 动态绑定在 Vue 中不能正常工作，因为 el-table 内部有自己的渲染机制。

**解决方案：使用 CSS 绝对定位实现视觉合并**

```vue
<template #default="scope">
  <div class="cell-wrapper">
    <!-- 已合并的行程块 -->
    <div 
      v-if="getMergedCell(scope.row, day) && getMergedCell(scope.row, day).startDay === day"
      class="merged-schedule-cell"
      :class="'purpose-' + getMergedCell(scope.row, day).purpose + ' status-' + getMergedCell(scope.row, day).status"
      :style="{ width: getMergedWidth(scope.row, day) }"
      @click.stop="handleScheduleClick(getMergedCell(scope.row, day))"
    >
      <el-tooltip :content="getScheduleTooltip(getMergedCell(scope.row, day))" placement="top">
        <span class="cell-text">{{ getMergedCell(scope.row, day).enterpriseName || getMergedCell(scope.row, day).userName }}</span>
      </el-tooltip>
    </div>
    
    <!-- 空白可选择的单元格 -->
    <div
      v-if="!getMergedCell(scope.row, day)"
      :class="['empty-cell', { 'selected-cell': isSelectedCell(scope.$index, day), 'rest-date': isRestDateForCell(scope.row, day) }]"
      @mousedown="handleCellMouseDown(scope.$index, scope.row, day)"
      @mouseup="handleCellMouseUp(scope.$index, scope.row, day)"
    />
  </div>
</template>
```

**CSS 样式：**
```css
.cell-wrapper {
  position: relative;
  width: 100%;
  height: 100%;
}

.merged-schedule-cell {
  position: absolute;
  top: 0;
  left: 0;
  height: 100%;
  display: flex;
  align-items: center;
  justify-content: center;
  cursor: pointer;
  color: white;
  font-size: 12px;
  border-radius: 4px;
  z-index: 2;
  padding: 2px 4px;
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

.merged-schedule-cell:hover {
  opacity: 0.9;
  box-shadow: 0 2px 8px rgba(0,0,0,0.15);
}

.empty-cell {
  width: 100%;
  height: 32px;
  cursor: crosshair;
  transition: all 0.15s ease;
}

.empty-cell:hover {
  background-color: #e6f7ff;
}

.selected-cell {
  background-color: #91d5ff !important;
  border: 2px dashed #1890ff;
}
```

**关键逻辑：**
```javascript
function getMergedWidth(row, day) {
  const merged = getMergedCell(row, day)
  if (merged && merged.startDay === day) {
    return (merged.span * 40) + 'px'  // 每天宽度约40px
  }
  return '0'
}
```

### 2.4 排班列表优化（按员工/企业分组合并）

**核心思路：** 将同一员工/企业的多天行程合并成一行显示

**数据处理函数：**
```javascript
function processScheduleList() {
  if (activeTab.value === 'employee') {
    const grouped = {}
    scheduleListData.value.forEach(item => {
      const key = item.userId + '_' + item.enterpriseId
      if (!grouped[key]) {
        grouped[key] = {
          userId: item.userId,
          userName: item.userName,
          enterpriseId: item.enterpriseId,
          enterpriseName: item.enterpriseName,
          purpose: item.purpose,
          status: item.status,
          dates: []
        }
      }
      grouped[key].dates.push(item.scheduleDate)
    })
    processedScheduleList.value = Object.values(grouped)
  } else {
    // 企业排班同理...
  }
}
```

**排班列表模板：**

### 2.5 拼音插件集成

**推荐使用：overtrue/pinyin**

1. 安装 Composer 包：
```bash
composer require overtrue/pinyin
```

2. 修改 BizEnterpriseService.php：
```php
use Overtrue\Pinyin\Pinyin;

public function getPinyin($text)
{
    $pinyin = new Pinyin();
    $result = $pinyin->abbr($text);  // 获取首字母
    return strtoupper($result);
}
```
